# 長篇小說快速預覽技能

## 核心理念
不需要讀完幾十萬字，通過智能採樣快速理解小說全貌。

## 採樣策略

### 三點採樣法
1. **開頭採樣** (前5000字)
   - 世界觀建立
   - 主角出場
   - 初始情境
   - 敘事風格

2. **中間採樣** (50%位置前後3000字)
   - 故事發展狀態
   - 關鍵轉折點
   - 角色成長
   - 衝突升級

3. **結尾採樣** (最後3000字)
   - 故事結局
   - 主題呈現
   - 情感收尾
   - 開放性問題

### 補充採樣（可選）
- **25%位置**: 第一幕結束
- **75%位置**: 高潮前夕

## 快速識別清單

### 從開頭識別
- [ ] 小說標題和類型
- [ ] 時代背景和地點
- [ ] 主角姓名和身份
- [ ] 初始困境/渴望
- [ ] 敘事視角（第一/第三人稱）
- [ ] 語言風格（華麗/簡潔/口語/文學）

### 從中間識別
- [ ] 主要衝突是什麼
- [ ] 反派/對手是誰
- [ ] 重要配角有哪些
- [ ] 故事走向（喜劇/悲劇/開放）

### 從結尾識別
- [ ] 故事如何收場
- [ ] 主角成長了什麼
- [ ] 主題是什麼
- [ ] 有無續集可能

## 集數預估公式

```
基礎集數 = 總字數 / 20000
調整係數:
  - 對話多 (+10%)
  - 動作多 (+15%)
  - 心理描寫多 (-10%)
  - 敘述密集 (-15%)

最終集數 = 基礎集數 × 調整係數
每集時長 = 10-15分鐘
```
