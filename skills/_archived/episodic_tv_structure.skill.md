# 連續劇敘事結構技能

> 📚 **TV Writing Books Collection**
> 參考：Pamela Douglas《Writing the TV Drama Series》
> 參考：Ellen Sandler《The TV Writer's Workbook》

## 連續劇 vs 單元劇

| 類型 | 定義 | 範例 |
|------|------|------|
| **單元劇** (Episodic) | 每集獨立故事 | CSI, 法律與秩序 |
| **連續劇** (Serialized) | 跨集連貫敘事 | 權力遊戲, 絕命毒師 |
| **混合型** (Hybrid) | 單集故事+長線發展 | X檔案, 超感神探 |

## 單集結構

### 傳統四幕式 (Cable/Network)
```
冷開場 (Teaser)     → 1-3分鐘，吸引觀眾
第一幕 (Act 1)      → 12-15分鐘，建立問題
第二幕 (Act 2)      → 12-15分鐘，複雜化
第三幕 (Act 3)      → 10-12分鐘，危機
第四幕 (Act 4)      → 8-10分鐘，高潮+結局
```

### 現代五幕式 (Streaming)
```
冷開場              → 2-5分鐘
第一幕              → 8-10分鐘
第二幕              → 10-12分鐘
第三幕              → 12-15分鐘
第四幕              → 10-12分鐘
第五幕 + 鉤子        → 5-8分鐘
```

### AI番劇結構 (10-15分鐘)
```
開場鉤子            → 30秒-1分鐘
情境建立            → 2-3分鐘
複雜化              → 4-5分鐘
危機決策            → 2-3分鐘
高潮+下集預告       → 2-3分鐘
```

## 多線敘事 (A/B/C Plot)

| 線 | 權重 | 功能 |
|----|------|------|
| **A線** | 50-60% | 主要故事，本集焦點 |
| **B線** | 25-30% | 次要故事，角色發展 |
| **C線** | 10-15% | 喜劇/輕鬆，調節節奏 |

### 交織方式
```
場景1: A線 - 發現屍體
場景2: B線 - 家庭衝突
場景3: A線 - 調查展開
場景4: C線 - 辦公室笑話
場景5: A+B線交匯 - 家人成為嫌疑人
...
```

## 季度弧線 (Season Arc)

### 10集季的結構
```
第1集: 引擎啟動    → 介紹世界+激勵事件
第2集: 深入        → 擴展問題+介紹配角
第3集: 複雜化      → 第一個轉折
第4集: 升級        → 提高賭注
第5集: 中點反轉    → 大揭示/假勝利
第6集: 反彈        → 處理中點後果
第7集: 危機來臨    → 一切開始崩壞
第8集: 黑暗時刻    → 最低點
第9集: 集結        → 準備最終戰役
第10集: 決戰       → 高潮+為下季埋線
```

### 鉤子設計 (Cliffhangers)

| 類型 | 說明 | 範例 |
|------|------|------|
| **懸念鉤** | 未解之謎 | 「他到底是誰？」|
| **危機鉤** | 危險即將發生 | 定時炸彈倒計時 |
| **揭示鉤** | 驚人真相 | 「他是你父親」|
| **情感鉤** | 關係轉折 | 分手/告白 |
| **死亡鉤** | 角色命運未定 | 槍響，畫面變黑 |

## 角色發展軌跡

### 季度角色弧線
```
主角:
  第1-3集: 建立初始狀態
  第4-6集: 受到挑戰
  第7-9集: 成長痛苦
  第10集: 完成蛻變
  
配角:
  每2-3集獲得焦點
  有自己的mini-arc
  與主角弧線呼應
```

## 長篇小說→連續劇映射

### 字數→集數公式
```
小說字數 / 15,000 = 基礎集數 (每集10-15分鐘)

調整係數:
- 對話密集 → +15%
- 動作密集 → +20%
- 內心描寫多 → -10%
- 敘述密集 → -15%
```

### 章節映射策略
```
短章節(3000字內) → 合併為1集
中章節(5000字) → 單獨1集
長章節(8000字+) → 拆分為2集
```
