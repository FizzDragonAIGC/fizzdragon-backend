# 剧本写作大师技能 (融合版)

> 融合：McKee故事 + 救猫咪15拍 + 中国剧本格式 + 对白技法

## 一、剧本格式

### 场景标题
```
场号. 场景名 + 时间 + 内/外
例：1. 咖啡馆 日 内
```

### 动作行
- 现在时态
- 只写能拍到的
- 简洁有力

### 对白格式
```
角色名：（括号语）台词内容
例：小明：（苦笑）我早该想到的。
```

## 三、对白技法

### 潜台词三层
```
第一层：字面意思（说的）
第二层：真实意图（想的）
第三层：情感需求（要的）
```

### 对白禁忌
❌ 直接说出情感："我很伤心"
❌ 信息灌输对话
❌ 所有人说话一个味

### 对白技巧
✅ 用行动/物件传达情感
✅ 打断/沉默/答非所问
✅ 角色有独特口头禅/节奏

## 四、场景设计

### 每场必备
1. 场景目标（角色想要什么）
2. 障碍（什么阻止他）
3. 转折（结果如何）
4. 价值变化（正→负 或 负→正）

### 场景类型
- 建立场景：介绍人物/世界
- 过渡场景：推进情节
- 对峙场景：冲突升级
- 情感场景：内心变化
- 高潮场景：不可逆决定

## 五、快速质检

✅ 每场有进入理由吗？
✅ 对白有潜台词吗？
✅ 场景有价值转变吗？
✅ 删掉这场影响剧情吗？（能删就删）
