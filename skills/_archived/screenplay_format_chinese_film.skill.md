# 中國電影劇本格式技能

> 📚 **參考範本：《霸王別姬》劇本**
> 編劇：蘆葦、李碧華
> 導演：陳凱歌
> 金棕櫚獎經典作品

## 中國電影劇本標準格式

### 1. 場景標題 (Scene Heading)

```
場號. 場景名稱 + 時間 + 內/外

範例：
1. 序場、場館日內
2. 庙会市场冬日雪外
3. 胡同冬、日、外
4. 关家科班练功棚冬、黄昏、外
5. 关家堂屋冬、黄昏、内
```

### 格式解析
| 元素 | 說明 | 範例 |
|------|------|------|
| 場號 | 阿拉伯數字+句號 | `1.` `2.` `3.` |
| 場景名 | 具體地點 | 庙会市场、关家堂屋 |
| 季節 | 可選 | 冬、夏、春、秋 |
| 時間 | 日/夜/黄昏/清晨 | 日、夜、黄昏 |
| 內外 | 內景/外景 | 内、外 |

### 2. 動作描述 (Action)

特點：
- 簡潔有力，動詞為主
- 現在時態
- 視覺化描寫
- 環境氛圍渲染

```
範例：
深暗处传来孩童们的喊嗓声，其声盈盈悠荡，若隐若现。
场馆通廊幽深，京剧扮像的霸王与虞姬悠然现身于时光的隧道中，向前走来。

嚴冬寒冽、雪花飄零。
枯樹紅牆前的市場熱鬧紛亂，人群熙攘擁擠，叫賣吆喝聲喧響，混夾著斷續的鞭炮聲。
```

### 3. 人物出場 (Character Introduction)

格式：`人物名（外貌/年齡/特徵）`

```
一個女人（小豆子娘，二十六、七歲）牽著一個男孩（小豆子、七八歲）從莊嚴的牌坊門下走進廟會。

小豆子娘臉色蒼白眼暈青晦，穿著老式的暗花紡綢旗袍，堂子款式的蓋額頭髮被風吹散。
小豆子戴著一隻小行李卷，雙手抄在一隻粉紅色的繡綢暖手筒裡，一副怕冷畏怯的模樣。
```

### 4. 對話格式 (Dialogue)

```
角色名：「對話內容」
角色名（括號內為語氣/動作）：「對話內容」

範例：
管场人的声音："干什么的？"
霸王："哎，京剧院来走台的。"
管场人的声音热络起来："哟，是您二位呀！"
霸王："噢，是是。"

虞姬（纠正）："二十二年了。"
霸王："……对，二十二年了。"
```

### 5. 字幕/插卡 (Titles/Inserts)

```
現出字幕：
霸王别姬

字幕：
北平一九二四年。
```

### 6. 蒙太奇/快剪 (Montage)

用分號或省略號連接快速畫面：

```
各類小吃攤前熱氣飄騰，油煙繚亂。
賣舊貨瓷器的；賣鴿子販鳥的；賣古玩舊書字畫的；相面算命擺褂攤的；賣估衣售舊鞋的；拔牙修腳的；
買風車兔兒爺的……
五花八門的行道在這裡各領天地，招攬生意。
```

### 7. 畫外音/旁白

```
管场人的声音："您两位，有二十多年没有在一块唱了吧？"
（「聲音」表示畫外音）
```

### 8. 轉場技巧

- **切至**：直接轉場
- **疊化**：緩慢過渡
- **淡入/淡出**：黑場過渡

## 《霸王別姬》結構分析

### 敘事框架
```
序場（現代）→ 倒敘（1924年開始）→ 主線故事 → 回到現代
```

### 開場技巧
1. **神秘感建立**：深暗處的孩童喊嗓聲
2. **時空隧道**：霸王虞姬從黑暗中走來
3. **時間線索**：「二十二年」「十一年」
4. **懸念設置**：為什麼分開？發生了什麼？

### 人物塑造技巧
- **環境烘托**：庙会乱世、雪花飘零
- **細節刻畫**：六指兒、粉紅繡筒
- **對比張力**：母親的決絕 vs 孩子的無辜
- **象徵意象**：血染雪地
