# Linda Seger《改編的藝術》技能

> 📚 **The Art of Adaptation: Turning Fact and Fiction into Film**
> 作者：Linda Seger（好萊塢頂級劇本顧問）

## 核心理念
「改編不是複製，而是翻譯——從一種媒介的語言翻譯成另一種媒介的語言。」

## 改編三大挑戰

### 1. 濃縮 (Condensation)
將長篇內容壓縮為影視時長

| 技巧 | 說明 | 範例 |
|------|------|------|
| **刪除副線** | 保留主線，砍掉次要情節 | 《魔戒》刪除Tom Bombadil |
| **合併角色** | 將多角色功能合一 | 三個朋友→一個閨蜜 |
| **壓縮時間** | 數月→數天 | 「三年後」跳躍 |
| **場景選擇** | 只留戲劇性場景 | 砍掉日常鋪墊 |

### 2. 擴展 (Expansion)
將短篇內容展開為影視長度

| 技巧 | 說明 |
|------|------|
| **增加副線** | 為配角創造獨立故事線 |
| **深化背景** | 擴展世界觀和歷史 |
| **增加衝突** | 添加障礙和對手 |
| **視覺化內心** | 將心理活動轉為外在行動 |

### 3. 翻譯 (Translation)
將文學語言轉為影視語言

| 文學技巧 | 影視轉化 |
|----------|----------|
| 內心獨白 | 畫外音/表情/行動 |
| 描述性段落 | 攝影/美術/聲音 |
| 時間跳躍 | 剪輯/過渡 |
| 象徵意象 | 視覺符號 |

## 分析源材料的5個問題

1. **什麼是核心故事？** 一句話概括
2. **主角的轉變弧線是什麼？** 從A→B的內在改變
3. **有哪些不可動搖的場景？** 觀眾期待的經典時刻
4. **什麼會抵抗改編？** 內心獨白、複雜時間線、大量角色
5. **原作的獨特風格是什麼？** 如何在影視中保留

## 長篇小說改編公式

```
原著字數 / 20,000 = 基礎集數
↓
調整係數（對話密集+10%，動作密集+15%）
↓
最終集數 × 45分鐘 = 總時長
```

## Seger改編清單

- [ ] 識別主線和可刪副線
- [ ] 列出必須保留的經典場景
- [ ] 找出可合併的相似角色

- [ ] 確定時間線壓縮/跳躍點
- [ ] 保留原作獨特風格元素
