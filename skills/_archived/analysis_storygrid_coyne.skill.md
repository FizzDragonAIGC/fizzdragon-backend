# Shawn Coyne《故事格子》分析技能

> 📚 **The Story Grid: What Good Editors Know (2015)**
> 作者：Shawn Coyne（25年出版業編輯）
> 特點：系統化的故事診斷工具

## 核心理念
「好故事必須達到某些客觀標準。這些標準可以被識別、測量和應用。」

## Story Grid 五大支柱

### 1. 類型 (Genre)
每個故事都有一個**主導類型**決定讀者期待

| 內容類型 | 核心價值 | 核心事件 | 讀者需求 |
|----------|----------|----------|----------|
| 動作 | 生/死 | 英雄犧牲 | 腎上腺素 |
| 恐怖 | 生/死 | 怪物襲擊 | 安全感 |
| 犯罪 | 正義/不義 | 真相揭露 | 公正感 |
| 愛情 | 愛/恨 | 戀人結合 | 親密感 |
| 成長 | 成熟/幼稚 | 覺醒時刻 | 意義感 |
| 世界觀 | 信仰/幻滅 | 啟示 | 理解 |

### 2. 五大組件 (Five Commandments)

每個場景必須包含：

```
1. 激勵事件 (Inciting Incident)
   ↓ 打破平衡
2. 進展複雜化 (Progressive Complications)
   ↓ 壓力升級
3. 危機 (Crisis)
   ↓ 兩難抉擇
4. 高潮 (Climax)
   ↓ 做出選擇
5. 結局 (Resolution)
   ↓ 新的平衡
```

### 3. 價值轉換 (Value Shift)

每個場景必須有**價值轉換**：
- 從正面→負面
- 從負面→正面
- 從正面→更正面
- 從負面→更負面

| 場景開始 | 場景結束 | 有效？ |
|----------|----------|--------|
| 安全 | 危險 | ✅ |
| 危險 | 危險 | ❌ 無變化 |
| 愛 | 恨 | ✅ |
| 恨 | 冷漠 | ✅ 負面→更負面 |

### 4. 故事脊椎 (Story Spine)

全局結構的15個里程碑：

```
開頭鉤子 → 激勵事件 → 漸進複雜1 → 漸進複雜2 
→ 第一幕轉折點 → 漸進複雜3 → 中點 → 漸進複雜4 
→ 危機 → 第二幕轉折點 → 漸進複雜5 → 高潮 
→ 結局 → 結尾 → 回聲
```

### 5. 電子表格分析

| 章節 | 場景 | 視角 | 事件 | 價值轉換 | 極性 | 筆記 |
|------|------|------|------|----------|------|------|
| 1 | 1 | 主角 | 日常生活 | 安全→威脅 | - | 建立 |
| 1 | 2 | 主角 | 發現屍體 | 無知→恐懼 | -- | 激勵 |
| 2 | 1 | 配角 | 調查開始 | 困惑→線索 | + | 進展 |

```

│ 高潮: _____________________________  │
```

## 長篇小說分析步驟

1. **識別類型** → 確定核心價值
2. **找出全局五組件** → 宏觀結構
3. **每章創建電子表格** → 微觀分析
4. **檢查價值轉換** → 確保每場有變化
5. **填寫Foolscap** → 一頁總結
