# Psychology & Motivation - 角色心理與動機設計

## 核心框架：Want / Need / Wound

### Want（表面慾望）
角色**意識到**並**主動追求**的目標
- 外在的、具體的、可衡量的
- 驅動情節前進
- 例：復仇、財富、愛情、權力、自由

### Need（深層需求）
角色**真正需要**但可能**不自知**的東西
- 內在的、抽象的、關於成長的
- 驅動角色弧線
- 例：被接納、自我價值、放下執念、學會信任

### Wound（心理創傷）
**過去的傷痛**塑造了現在的行為模式
- 童年經歷、失去、背叛、失敗
- 解釋角色的恐懼和防禦機制
- 是 Need 形成的原因

---

## 動機公式

```
Wound (過去) → Fear (現在的恐懼) → Want (錯誤的解決方案) → Need (真正的解答)
```

### 範例

**復仇者角色**
- Wound: 父親被殺
- Fear: 無力保護所愛之人
- Want: 殺死仇人（錯誤解答）
- Need: 放下仇恨，學會向前看（真正解答）

**野心家角色**
- Wound: 童年貧困被歧視
- Fear: 再次變得軟弱無力
- Want: 無上權力和財富（錯誤解答）
- Need: 接納自己的過去，不需要外在證明（真正解答）

---

## 角色原型與動機

| 原型 | 典型 Want | 典型 Need | 典型 Wound |
|------|-----------|-----------|------------|
| 英雄 | 拯救世界 | 相信自己 | 被質疑/失敗經歷 |
| 導師 | 傳承智慧 | 放手讓徒弟成長 | 過去的失敗學生 |
| 陰影 | 毀滅/控制 | 被理解/療癒 | 深重的背叛或失去 |
| 使者 | 完成使命 | 找到自我認同 | 失去歸屬感 |
| 變形者 | 保護自己 | 選擇立場 | 信任被背叛 |
| 盟友 | 幫助主角 | 被認可自己的價值 | 被忽視的經歷 |
| 門檻守衛 | 阻擋入侵 | 找到新的使命 | 曾經的失職 |

## 心理深度層級

### Level 1: 表面行為
觀眾直接看到的動作和對話

### Level 2: 表面動機
角色說出來的理由
- "我要報仇"
- "我想保護家人"

### Level 3: 真實動機
角色可能不會說但我們理解的
- 害怕再次失去
- 需要證明自己的價值

### Level 4: 無意識驅力
角色自己都不知道的深層原因
- 童年被拋棄的創傷
- 對父親認可的渴望

---

## 衝突設計

### 內在衝突
Want vs Need 的拉扯
- 想要復仇（Want）但內心渴望平靜（Need）
- 追求權力（Want）但真正需要愛（Need）

### 外在衝突
與其他角色的對立
- 目標衝突：都想要同一樣東西
- 價值觀衝突：對錯的定義不同
- 方法衝突：目標相同但手段不同

## 角色弧線設計

### 正向弧線（成長）
```
開始：被 Wound 控制，追求錯誤的 Want
中間：遭遇挫折，開始質疑
轉折：意識到真正的 Need
結束：放下 Want，擁抱 Need，療癒 Wound
```

### 負向弧線（墮落）
```
開始：有正當的追求
中間：遭遇誘惑或挫折
轉折：做出錯誤選擇
結束：完全被 Wound 控制，成為反派
```

### 平坦弧線（堅持）
```
角色不變，但改變周圍的世界
- 用於導師型或象徵型角色
- 角色的信念被考驗但最終堅持
```
