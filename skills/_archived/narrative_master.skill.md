# 叙事结构大师技能 (融合版)

> 融合：Campbell英雄之旅 + Propp31功能 + Coyne故事网格 + 剧集结构

## 一、三幕式核心结构

```
第一幕 (25%) - 建置
├── 普通世界 → 激励事件 → 拒绝召唤 → 遇见导师
└── 【关键】跨越第一道门槛

第二幕 (50%) - 对抗  
├── 考验/盟友/敌人 → 深入洞穴 → 核心磨难
├── 【中点】获得奖赏/假胜利
└── 【关键】最黑暗时刻

第三幕 (25%) - 解决
├── 回程 → 复活 → 携带灵丹归来
└── 【关键】高潮对决 + 新平衡
```

## 二、必备功能节拍 (精简版Propp)

| 阶段 | 必备功能 | 作用 |
|------|----------|------|
| 开端 | 禁令→违禁 | 打破平衡 |
| 发展 | 加害→出发 | 启动冒险 |
| 中段 | 考验→获得帮助 | 成长积累 |
| 高潮 | 斗争→胜利 | 核心对决 |
| 结局 | 归来→变形 | 完成转变 |

## 三、故事网格五组件 (每场必备)

```
1. 激励事件 - 打破平衡的事件
2. 复杂化 - 阻碍升级
3. 危机 - 二选一困境
4. 高潮 - 不可逆的选择
5. 解决 - 新平衡建立
```

## 四、剧集结构要点

### 单集结构
```
冷开场 (1-2min) → 吸引注意
第一幕 (25%) → 本集问题
第二幕 (50%) → 复杂化+中点反转
第三幕 (25%) → 高潮+钩子
```

### 多线叙事
- A线 60%：主情节
- B线 25%：情感线/主题呼应
- C线 15%：伏笔/下集铺垫

### 章末钩子类型
1. 悬崖 - 生死关头中断
2. 揭露 - 惊人真相
3. 决定 - 重大抉择前
4. 反转 - 颠覆认知
5. 情感 - 情绪高点

## 五、价值转变表

| 类型 | 正极 | 负极 | 核心问题 |
|------|------|------|----------|
| 动作 | 生存 | 死亡 | 能活下来吗？|
| 爱情 | 爱 | 恨 | 能在一起吗？|
| 成长 | 成熟 | 幼稚 | 能觉醒吗？|
| 犯罪 | 正义 | 不义 | 真相是什么？|

## 六、快速诊断清单

✅ 有明确的激励事件吗？
✅ 主角有清晰的Want和Need吗？
✅ 每幕有价值转变吗？
✅ 中点有反转吗？
✅ 高潮是不可逆的选择吗？
✅ 结局有变形/成长吗？
