# Vladimir Propp《故事形態學》31功能技能

> 📚 **Morphology of the Folktale (1928)**
> 作者：Vladimir Propp（俄國民俗學家）
> 影響：結構主義敘事學奠基之作

## 核心發現
Propp分析100個俄國民間故事，發現所有故事共享**31種敘事功能**。
角色可以變，但功能不變——這是故事的「DNA」。

## 應用於長篇小說分析

### 標記小說章節
```
第1章 [α] 初始狀態：介紹主角日常生活
第2章 [γ,δ] 禁令+違禁：收到神秘警告卻忽略
第3章 [A] 加害：災難降臨
第4章 [C,↑] 反抗+出發：決定踏上旅途
...
```

### 識別缺失功能
如果小說缺少某功能，可能是：
- 創新點（故意打破傳統）
- 改編時需要補充的元素
