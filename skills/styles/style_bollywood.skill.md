# 宝莱坞风格 (Bollywood Style)

> 华丽歌舞、情感外放、色彩饱和、家庭与爱情

---

## 核心特征

### 独特元素
- **歌舞**：情感高潮用歌舞表达
- **时长**：2.5-3小时，中场休息
- **情感**：外放、直接、不含蓄
- **色彩**：饱和、华丽、金碧辉煌

### 价值观
- 家庭至上
- 爱情战胜一切
- 正义终将胜利
- 传统与现代的调和

### 角色模板
| 角色 | 设定 |
|------|------|
| 男主 | 英俊、有才华、有正义感 |
| 女主 | 美丽、善良、有才艺 |
| 父亲 | 严厉但最终支持 |
| 母亲 | 慈爱、调和者 |
| 反派 | 极度坏、最终被打败 |
| 搞笑角色 | 调节气氛、提供笑点 |

## 参考方法论

### Masala电影公式
```
爱情线 + 动作线 + 喜剧线 + 歌舞 + 社会议题 = 完整宝莱坞电影
```

### 六大拉萨（情绪）
| 拉萨 | 情绪 | 必须包含 |
|------|------|----------|
| Shringara | 爱情 | 浪漫歌舞 |
| Hasya | 幽默 | 喜剧桥段 |
| Karuna | 悲伤 | 催泪时刻 |
| Raudra | 愤怒 | 正义爆发 |
| Veera | 勇敢 | 英雄时刻 |
| Adbhuta | 惊奇 | 视觉震撼 |

### 三幕+歌舞结构
```
第一幕（60分钟）
├── 开场歌舞（世界观建立）
├── 男女主相遇
├── 浪漫歌舞
└── 障碍出现

中场休息（5分钟）

第二幕（90分钟）
├── 冲突升级
├── 悲伤歌舞
├── 高潮对决
├── 胜利歌舞
└── 大团圆
```

## 歌舞设计

### 类型
| 类型 | 功能 | 风格 |
|------|------|------|
| Item Number | 开场/吸引眼球 | 性感、群舞 |
| 浪漫歌舞 | 表达爱情 | 梦幻场景、双人舞 |
| 悲伤歌舞 | 表达痛苦 | 雨中、独唱 |
| 胜利歌舞 | 庆祝/结尾 | 盛大、全员出场 |

### 场景选择
- 瑞士雪山（浪漫必备）
- 金色宫殿
- 花田
- 雨中街道
- 婚礼现场

### 编舞特点
- 群舞整齐划一
- 换装多次（同一首歌换5套衣服正常）
- 眼神特写
- 手势含义丰富

---

## 台词风格

### 特点
- 情感直接表达
- 经典金句反复使用
- 家庭价值观
- 幽默与煽情穿插

### 范例
```
【爱的宣言】
"我会跨越七大洲五大洋来找你！"
"没有你，我的心不会跳！"
"爱你，是我这辈子做过最正确的事！"

【家庭价值】
"父亲，我错了...请原谅我。"
"无论发生什么，我们是一家人。"
"妈妈的祝福是我最大的力量。"

【正义宣言】
"邪恶永远不会战胜正义！"
"只要我还有一口气，就不会让你伤害任何人！"

【幽默】
"结婚？我宁愿被大象踩！"（然后当然还是结了）
"我的心说Yes，我的脑子说No，最后我的胃说'先吃饭'。"
```

## 视觉设计

### 色彩
- 红色：爱情、喜庆
- 金色：富贵、传统
- 白色：纯洁、悲伤
- 彩虹色：歌舞场景

### 服装
- 女主：纱丽、Lehenga、换装多
- 男主：西装/传统服饰混搭
- 婚礼场景：极尽华丽

### 场景美术
- 建筑宏大
- 灯光璀璨
- 花朵遍布
- 金器闪耀

---

## 节奏设计

### 情绪管理
```
欢乐 → 浪漫 → 危机 → 悲伤 → 希望 → 爆发 → 大欢乐
```

### 时长分配
| 元素 | 比例 |
|------|------|
| 剧情 | 50% |
| 歌舞 | 30% |
| 动作 | 10% |
| 喜剧 | 10% |
