# 治愈日常风格 (Iyashikei Style)

> 慢节奏、温暖细节、没有坏人、小确幸

---

## 核心特征

### 精神内核
- **治愈**：让观众感到平静和温暖
- **日常**：平凡生活中的美好
- **缓慢**：不急不躁，享受当下
- **善良**：世界上没有真正的坏人

### 氛围关键词
- 温柔、宁静、舒适、怀旧
- 自然、季节、慢时光
- 小事情、大温暖

### 角色特点
| 特点 | 说明 |
|------|------|
| 善良 | 每个角色都有好的一面 |
| 缺陷 | 有小毛病但不是坏人 |
| 成长 | 慢慢变好，不是戏剧性转变 |
| 关系 | 温暖的人际连接 |

---

## 参考方法论

### 小津安二郎美学
- 固定机位、低角度
- 日常对话、家庭餐桌
- 季节流转、时间感
- 不说破、留给观众体会

### 是枝裕和手法
- 非职业演员的自然表演
- 即兴对话、真实反应
- 食物戏（吃饭=情感交流）
- 长镜头、生活流

### 宫崎骏治愈元素
- 自然风景的细致描绘
- 食物的诱人呈现
- 飞行/旅途的自由感
- 温暖的避风港

---

## 叙事结构

### 非冲突式叙事
```
传统剧本：冲突→升级→高潮→解决
治愈系：日常→小事件→感悟→继续日常
```

### 季节结构
| 季节 | 氛围 | 常见情节 |
|------|------|----------|
| 春 | 开始、希望 | 相遇、新生活 |
| 夏 | 热情、回忆 | 祭典、花火、旅行 |
| 秋 | 收获、感伤 | 团聚、丰收 |
| 冬 | 温暖、反思 | 围炉、年终 |

### 单集结构
```
开场：平静的日常（30秒）
↓
小事件：发生了一件小事
↓
应对：角色用温和的方式处理
↓
感悟：小小的心灵触动
↓
收尾：回归日常，但有了微小变化
```

---

## 台词风格

### 特点
- 简单、自然、不文艺
- 留白比说话重要
- 食物相关的对话
- 四季和天气

### 范例
```
【日常对话】
"今天的天气真好呢。"
"是啊...适合晒被子。"
（两人一起看天空，不说话）

【关心】
"吃了吗？"
"还没。"
"那...一起吃吧。"

【感悟】
"原来...这就是幸福吧。"
"嗯...也许吧。"

【季节感】
"樱花开了呢。"
"明年这个时候，我们还会在一起看吗？"
"...会的吧。"

【食物】
"好吃。"
"真的吗？太好了。"
（安静地继续吃）
```

## 视觉设计

### 色调
- 柔和、低饱和度
- 暖黄（室内、回忆）
- 淡蓝（天空、海边）
- 绿色（自然、治愈）

### 场景
| 场景 | 氛围 |
|------|------|
| 家 | 温暖的避风港 |
| 街道 | 熟悉的小镇 |
| 自然 | 山、海、田野 |
| 学校 | 青春、纯真 |
| 小店 | 人情味 |

### 镜头语言
- 长镜头（让观众沉浸）
- 空镜头（风景、静物）
- 固定机位（稳定感）
- 自然光（柔和）

---

## 节奏设计

### 慢节奏要素
- 不急着推进情节
- 允许沉默和留白
- 重复的日常仪式
- 季节流转的时间感

### 情绪曲线
```
平静 → 微波澜 → 平静 → 小感动 → 平静 → 温暖
```

### 音乐
- 钢琴、吉他为主
- 自然音效（风声、虫鸣）
- 轻柔、不喧宾夺主

---

## 关键元素

### 必须有的
- 食物戏（认真做饭/吃饭）
- 季节感（樱花、枫叶、雪）
- 温暖的关系
- 小动物（猫特别重要）
- 善意的人

### 不能有的
- ❌ 真正的坏人
- ❌ 激烈的冲突
- ❌ 悲惨的结局
- ❌ 快节奏剪辑
- ❌ 过度戏剧化

---
