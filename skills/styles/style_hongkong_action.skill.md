# 港片黄金时代风格 (Hong Kong Golden Era Style)

> 江湖义气、兄弟情、宿命悲剧、枪战美学

---

## 核心特征

### 精神内核
- **义气**：朋友比天大
- **宿命**：江湖路没有退路
- **悲壮**：英雄末路的美学
- **情义**：哪怕敌对也惺惺相惜

### 两大类型
| 类型 | 代表 | 特点 |
|------|------|------|
| **英雄片** | 吴宇森 | 白鸽教堂、双枪慢镜、兄弟情 |
| **武侠片** | 徐克 | 飞檐走壁、剑气如虹、江湖恩怨 |

### 角色模板
| 角色 | 设定 |
|------|------|
| 大哥 | 讲义气、扛事、有人格魅力 |
| 兄弟 | 忠诚、为大哥赴汤蹈火 |
| 叛徒 | 曾经的兄弟、利益面前变节 |
| 条子 | 警匪边界模糊、亦正亦邪 |
| 女人 | 红颜知己、知道男人难当 |
| 老大 | 幕后boss、终极对决对象 |

---

## 参考方法论

### 吴宇森暴力美学
- **慢镜头**：子弹轨迹、飞鸽、火光
- **双枪**：左右开弓，仪式感
- **白衣/黑衣**：正邪对立的视觉化
- **教堂/医院**：神圣场所的血战
- **对视**：杀前的惺惺相惜

### 黑帮片结构
```
兄弟结义 → 共创江山 → 利益分歧 → 背叛危机 → 血战收场
```

### 武侠片结构
```
出山入世 → 恩怨纠葛 → 高手过招 → 绝学对决 → 归隐/死战
```

## 台词风格

### 特点
- 简洁有力，不废话
- 江湖黑话
- 金句多，适合海报
- 生死看淡，不装逼

### 范例
```
【义气】
"我当你是兄弟！"
"今日你帮我，日后有事，一声。"
"一日兄弟，一世兄弟。"

【宿命】
"人在江湖，身不由己。"
"出来混，迟早要还的。"
"这条路，走了就回不了头。"

【悲壮】
"大不了一死，怕什么！"
"做人最紧要信字。"
"有些事，不做就不是人。"

【警匪】
"你是警察，我是贼，咱们井水不犯河水。"
"做条子，做古惑仔，都是讨生活。"

【临死】
"我只后悔...没早点杀你。"
"下辈子...别做兄弟了。"
"帮我照顾好她。"
```

## 视觉设计

### 场景
| 场景 | 功能 |
|------|------|
| 茶餐厅 | 江湖议事、日常 |
| 码头/仓库 | 交易、火拼 |
| 夜总会 | 大佬出场、权力象征 |
| 天台 | 对峙、心事独白 |
| 雨中街道 | 追杀、宿命感 |

### 动作美学
- **枪战**：双枪、翻滚、慢镜、火花
- **肉搏**：拳拳到肉、不花哨
- **追车**：窄巷穿行、撞击
- **刀战**：一刀见血、干净利落

### 色调
- 霓虹灯光（夜场）
- 暖黄（回忆/兄弟情）
- 冷蓝（危机/杀戮）
- 红色（血/警示）

---

## 节奏设计

### 港片速度
- 开场就要抓人（不废话）
- 每10分钟一个高潮
- 台词密度高，信息量大
- 动作戏紧凑

### 情绪曲线
```
义气建立 → 小危机 → 更义气 → 大背叛 → 血战 → 悲壮收尾
```

### 单场设计
```
对话铺垫 (1分钟) → 气氛紧绷 (30秒) → 突然爆发 (动作) → 收尾金句
```
