# 赛博朋克风格 (Cyberpunk Style)

> 高科技低生活、霓虹雨夜、身份认同、反乌托邦

---

## 核心特征

### 世界观设定
- **高科技**：AI、义体改造、虚拟现实、黑客
- **低生活**：贫民窟、底层挣扎、社会分化
- **大公司**：企业统治一切，比政府更强
- **边缘人**：黑客、雇佣兵、赏金猎人、走私者

### 哲学主题
| 主题 | 探讨问题 |
|------|----------|
| 身份 | 什么定义了"我"？记忆？身体？灵魂？|
| 自由 | 在监控社会中真的有自由吗？|
| 人性 | 机器能比人类更"人性"吗？|
| 现实 | 虚拟与现实的边界在哪里？|
| 阶级 | 技术进步是否加剧不平等？|

### 角色类型
| 角色 | 设定 |
|------|------|
| 黑客 | 信息就是武器，挑战系统 |
| 雇佣兵 | 为钱做任何事，有自己的底线 |
| 义体人 | 身体大部分是机械，身份认同危机 |
| AI | 觉醒的人工智能，追寻存在意义 |
| 企业人 | 系统的一部分，或成为叛徒 |
| 街头小贩 | 底层视角，人情味 |

## 参考方法论

### William Gibson《神经漫游者》
- "天空是电视调成死频道的颜色"
- 赛博空间的视觉化描写
- 技术与毒品的并置
- 边缘人的生存智慧

### Philip K. Dick
- 什么是真实？什么是人？
- 记忆的不可靠性
- 共情能力定义人性

### 《银翼杀手》美学
- 霓虹+雨+黑暗
- 东方元素混搭
- 废墟中的奢华
- 复制人的眼泪

---

## 台词风格

### 特点
- 技术术语与街头俚语混合
- 存在主义式独白
- 黑色幽默
- 哲学追问

### 范例
```
【技术+街头】
"我入侵了他们的冰墙，十秒搞定。代价？我左眼的神经接口烧了。"
"这个世界没有黑客，只有还没被企业收编的自由人。"

【身份追问】
"我的记忆是真的吗？还是某个实验室里植入的数据？"
"如果我的身体99%是机械，那个1%的肉体还是'我'吗？"
"她死了...但她的备份还在。那个备份，还是她吗？"

【反乌托邦】
"在这个城市，你不是消费者就是商品。"
"自由？自由是富人才买得起的奢侈品。"
"大公司不需要控制你的身体，他们只需要控制你的信用分。"

【黑色幽默】
"好消息是你还活着。坏消息是你欠了三家公司的债。"
"死亡不是终点，只要你买了灵魂备份服务。"
```

## 视觉设计

### 色彩
- **主色**：霓虹蓝、霓虹粉、黄绿
- **辅色**：深黑、钢铁灰
- **对比**：黑暗中的霓虹刺目

### 场景
| 场景 | 氛围 |
|------|------|
| 贫民窟 | 拥挤、霓虹、雨水、电线 |
| 企业总部 | 极简、白色、冷酷、监控 |
| 黑市 | 地下、混乱、各种改造品 |
| 虚拟空间 | 几何、数据流、超现实 |
| 义体诊所 | 手术台、零件、非法改造 |

### 气象
- 永远在下雨
- 雾气弥漫
- 人工光源（没有阳光）
- 飞行器穿梭

---

## 节奏设计

### noir式结构
```
委托/发现 → 调查/追踪 → 陷阱/背叛 → 真相/代价
```

### 动作节奏
- 黑客入侵：紧张、时间压力
- 街头追逐：快速剪辑
- 义体战斗：超人类速度
- 哲学对话：慢、留白
