# 昆汀式风格 (Tarantino Style)

> 话痨对白、章节跳跃、暴力美学、流行文化致敬

---

## 核心特征

### 标志性元素
- **话痨对白**：角色可以聊5分钟汉堡包
- **非线性**：章节跳跃、打乱时间线
- **突然暴力**：聊着聊着突然开枪
- **流行文化**：引用电影、音乐、漫画
- **复仇主题**：爽快的复仇叙事

### 叙事特点
- 章节标题（Chapter 1, 2, 3...）
- 时间线打乱
- 长对话场景
- 紧张气氛骤然爆发
- 黑色幽默

### 角色特点
| 特点 | 说明 |
|------|------|
| 话痨 | 每个角色都有独特的说话方式 |
| 酷 | 即使是反派也很有魅力 |
| 专业 | 各自领域的专家 |
| 有品味 | 对音乐/电影/食物有讲究 |

## 台词风格

### 特点
- 絮絮叨叨但有节奏
- 看似无关实则铺垫
- 每个角色声音不同
- 充满流行文化引用

### 范例
```
【经典开场（看似无关）】
A: "你知道他们在巴黎怎么叫四分之一磅汉堡吗？"
B: "不知道。"
A: "皇家芝士堡。因为他们用公制..."
B: "皇家芝士堡..."
A: "没错。那你知道他们怎么叫巨无霸吗？"
B: "不叫巨无霸？"
A: "不，叫'Le Big Mac'。但他们发音是法语的..."
（聊了5分钟汉堡后突然拔枪）

【紧张对峙】
"我要问你一个问题。你的回答决定你今晚能不能活着走出这扇门。"
（漫长的沉默）
"你准备好了吗？"

【突然暴力】
A: "所以你是说—"
（砰！A被爆头）
B: "...真糟糕。"
C: "他说什么了？"
B: "谁知道呢。"

【复仇宣言】
"你们欠我一条命。我来收债了。"
"当你收拾好你那破碎的灵魂，如果你还能爬，就来找我。"
```

## 结构设计

### 章节式
```
Chapter 1: 标题（一个场景/一组角色）
Chapter 2: 标题（另一个时间线/视角）
Chapter 3: 标题（回到过去）
...
Chapter N: 标题（所有线索交汇）
```

### 时间线游戏
- 先展示结果，再解释过程
- 多视角讲同一事件
- 闪回揭示关键信息
- 死人可以先出场

### 场景设计
```
长对话（5-10分钟） → 气氛紧绷 → 突然爆发 → 短暂暴力 → 冷幽默收尾
```

---

## 暴力美学

### 原则
- 暴力是标点符号，不是文章本身
- 突然、干脆、不拖泥带水
- 有时候故意拖长（折磨观众）
- 血浆可以夸张

### 设计
- 对峙时间 >> 动作时间
- 聊天中突然开枪
- 慢镜头偶尔使用
- 配乐反差（欢快音乐+暴力场面）

---

## 配乐哲学

### 特点
- 冷门老歌（60-70年代）
- 音乐与画面反差
- 场景记忆点
- 绝对不用原创配乐

### 参考
- Surf Rock
- Soul / Funk
- 意大利西部片配乐
- 日本动作片主题曲
