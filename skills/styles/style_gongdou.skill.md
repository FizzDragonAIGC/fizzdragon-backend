## 核心特征

### 精神内核
- **权**：权力是唯一的货币
- **谋**：每一步都是棋局
- **忍**：能忍常人所不能忍
- **狠**：该出手时绝不手软

## 台词风格

### 特点
- 一语双关（明一层暗一层）
- 绵里藏针（笑着骂人）
- 以退为进
- 借力打力

### 范例
```
【绵里藏针】
"妹妹年纪小，不懂事，本宫不与你计较。"
（暗讽：你就是不懂规矩）

"姐姐这身衣裳真好看，可惜料子旧了些。"
（暗讽：你失宠了）

【以退为进】
"臣妾不敢。"
（潜台词：但臣妾做了）

"嫔妾愚钝，不懂姐姐的意思。"
（潜台词：我装傻，你能怎样？）

【借力打力】
"皇上，臣妾只是担心姐姐身子骨受不住..."
（用关心的名义告状）

【威胁】
"本宫劝你想清楚。这宫里，有些话说出去，就收不回来了。"

【黑化宣言】
"从今以后，我不会再心软。"
"既然她们不给我退路，我也不必留手。"
```

## 结构设计

### 宫斗升级路线
```
入宫（白莲花）
↓
受欺（积累仇恨）
↓
觉醒（开始反击）
↓
第一次胜利（扳倒小角色）
↓
危机（被反派陷害）
↓
绝地反击（大计谋）
↓
上位（获得实权）
↓
终极对决（与大boss决战）
↓
结局（黑化/释然/死亡）
```

### 单集结构
```
开场：问安/日常（暗流涌动）
↓
事件：有人使坏/机会出现
↓
应对：计谋展开
↓
交锋：台词battle
↓
结果：本集谁赢谁输
↓
伏笔：更大阴谋
```

## 计谋设计

### 常用套路
| 计谋 | 说明 |
|------|------|
| 借刀杀人 | 让别人替自己出手 |
| 反间计 | 离间敌人内部 |
| 苦肉计 | 伤害自己陷害对方 |
| 调虎离山 | 转移注意力 |
| 欲擒故纵 | 假装不在意 |
| 将计就计 | 顺着对方的计中计 |

### 证据与把柄
- 信物：香囊、玉佩、书信
- 证人：宫女、太监、�的弟子
- 现场：布置陷阱
- 时机：节庆、生辰、选秀
