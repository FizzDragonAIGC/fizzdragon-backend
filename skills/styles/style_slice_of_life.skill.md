# 日常生活敘事風格 (Slice of Life)

> 📚 **核心參考**
> - 《深夜食堂》- 人情溫暖
> - 《小森林》- 療癒治愈
> - 《海街日記》- 家庭日常
> - 是枝裕和作品 - 生活質感

---

## 一、核心特徵

### 日常敘事的本質
> "最動人的故事，往往藏在最平凡的日子裡。"

**三大支柱：**
1. **生活質感**：真實的細節、熟悉的場景
2. **人情冷暖**：鄰里關係、家庭羈絆
3. **淡淡餘韻**：沒有大起大落，但有回味

---

## 二、對白風格

### 示範對白

**家庭日常：**
```
「飯做好了。」
「嗯。」
「今天加了點蔥花，你小時候最愛吃的那種。」
「...媽，我都三十了。」
「三十歲就不能愛吃蔥花了？」
```

**鄰里溫情：**
```
「王奶奶，今天的蔬菜特新鮮。」
「小李啊，你又瘦了，是不是又忘了吃早飯？」
「忙嘛。」
「忙也要吃飯。來，拿兩個饅頭，奶奶自己蒸的。」
```

**獨處時刻：**
```
「一個人吃火鍋，其實也沒那麼孤單。
畢竟，可以全部點自己愛吃的。」
```

---

## 三、結構設計

### 日常敘事節奏
```
建立日常(20%) → 小波折出現(20%) → 
相處/思考(30%) → 小小領悟(20%) → 
回歸日常(10%)
```

### 特點
- **無大衝突**：不需要反派或危機
- **情緒平穩**：波動小但細膩
- **結局開放**：生活繼續，故事留白
- **季節感**：春夏秋冬的循環

---

## 四、關鍵元素

### 生活場景
| 場景 | 功能 |
|------|------|
| 廚房/餐桌 | 家庭羈絆、食物治愈 |
| 街角小店 | 鄰里人情、城市記憶 |
| 公園長椅 | 偶遇、思考、獨處 |
| 回家的路 | 過渡、獨白、季節感 |
| 窗邊 | 觀察、發呆、時間流逝 |

### 情感表達方式
- 不說「我愛你」，用行動表達
- 不說「謝謝」，用沉默傳遞
- 爭吵不激烈，冷戰更常見
- 和解不煽情，一杯茶就夠

---

## 五、視覺設計

### 色調建議
| 類型 | 主色調 | 氛圍 |
|------|--------|------|
| 溫暖治愈 | 暖黃 | 家的溫度 |
| 清新自然 | 草木綠 | 療愈放鬆 |
| 懷舊感 | 褪色暖 | 時間質感 |
| 都市孤獨 | 灰藍 | 疏離+溫暖 |

### 標誌性畫面
- 冒著熱氣的食物
- 陽光透過窗簾
- 街道上的行人
- 夕陽下的剪影
- 雨後的街道反光
