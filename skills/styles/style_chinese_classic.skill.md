# 中式叙事风格 (Chinese Classic Style)

> 起承转合、意境留白、诗意表达、含蓄内敛

---

## 核心特征

### 结构规则（起承转合）
| 段落 | 比例 | 功能 | 情绪 |
|------|------|------|------|
| **起** | 20% | 开篇立意、人物登场、世界展现 | 平静→好奇 |
| **承** | 30% | 承接发展、关系深化、矛盾酝酿 | 渐入佳境 |
| **转** | 30% | 情节转折、高潮迭起、命运抉择 | 波澜壮阔 |
| **合** | 20% | 收束全篇、余韵悠长、意在言外 | 释然/怅然 |

### vs 好莱坞三幕式
| | 好莱坞 | 中式 |
|---|--------|------|
| 节奏 | 紧凑、高效 | 舒缓、留白 |
| 表达 | 直接说出 | 含蓄藏起 |
| 高潮 | 爆发、对决 | 内敛、悲凉 |
| 结局 | 大团圆 | 余韵/开放 |
| 主题 | 说出来 | 悟出来 |

### 审美特质
- **意境留白**：不说透，让观众自己悟
- **借景抒情**：景即是情，情景交融
- **含蓄内敛**：台词少说，眼神传情
- **因果轮回**：善恶有报，天道循环
- **家国情怀**：小我服从大我
- **天人合一**：自然与人物情绪呼应

## 参考方法论

### 传统叙事智慧
- **《文心雕龙》**：风骨、情采、隐秀
- **戏曲结构**：引子→正戏→收尾
- **章回体**：回目对仗、悬念钩子

### 情感表达
- **赋**：直接描述
- **比**：以彼喻此
- **兴**：先言他物引起所咏

### 人物塑造
- 性格通过行动展现，不用旁白解释
- 留有道德灰度，不做非黑即白
- 人物命运与时代命运交织

---

## 台词风格

### 特点
- 言有尽而意无穷
- 对话中见人物性格
- 引经据典但不掉书袋
- 重要时刻用沉默

### 范例
```
【含蓄表达爱意】
❌ "我爱你"
✅ "若你安好，我便是晴天"
✅ （递伞）"外面雨大"

【表达离别】
❌ "我很伤心你要走"
✅ "长亭外，古道边..."
✅ （望着远方）"风起了"

【表达决心】
❌ "我一定会成功"
✅ "不到黄河心不死"
✅ "此去经年，应是良辰好景虚设"

【表达释然】
❌ "我已经放下了"
✅ "也罢"
✅ （笑）"人生如戏"
```

### 对白技巧
- 一语双关（明说A暗指B）
- 借古讽今
- 欲言又止
- 此时无声胜有声

## 视觉/意境设计

### 景物对应情绪
| 景物 | 情绪 |
|------|------|
| 落花 | 离别、青春逝去 |
| 明月 | 思念、团圆渴望 |
| 流水 | 时光、命运不可逆 |
| 秋风 | 萧瑟、英雄迟暮 |
| 春雨 | 希望、新生 |
| 雪夜 | 孤独、清冷 |
| 红梅 | 坚韧、傲骨 |

### 色彩美学
- 水墨留白：大面积空白营造意境
- 红与黑：喜庆/悲剧的经典对比
- 青绿山水：田园理想

### 空间叙事
- 庭院：家族/权力中心
- 山水：出世/隐逸
- 长街：世俗人生
- 桥：过渡/抉择
- 亭：等待/送别

## 写作检查清单

✅ 用起承转合而非三幕式结构吗？
✅ 有借景抒情的段落吗？
✅ 台词足够含蓄吗？
✅ 主题是"悟出来"而非"说出来"吗？
✅ 结局有余韵吗？
✅ 人物行动而非说教展现性格吗？
✅ 天人合一：自然配合情绪了吗？
