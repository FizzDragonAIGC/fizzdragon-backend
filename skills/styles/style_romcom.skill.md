# 浪漫喜剧风格 (Romantic Comedy Style)

> 欢喜冤家、误会与和解、轻松幽默、甜蜜结局

---

## 核心特征

### 爱情公式
- **Meet-Cute**：可爱/有趣的相遇方式
- **障碍**：为什么不能在一起
- **发展**：从讨厌到喜欢
- **黑暗时刻**：分手/误会
- **Grand Gesture**：浪漫的挽回
- **HE**：在一起

### 喜剧元素
- 误会产生笑料
- 尴尬场面
- 配角搞笑
- 自嘲幽默

### 角色设计
| 角色 | 常见设定 |
|------|----------|
| 女主 | 事业型/笨拙/有追求 |
| 男主 | 表面不靠谱但真诚/霸道但善良 |
| 闺蜜 | 给建议/制造笑料/说真话 |
| 兄弟 | 鼓励男主/搞笑 |
| 情敌 | 表面完美但内心有问题 |
| 老板/家长 | 制造压力 |

## 台词风格

### 特点
- 机智、快速
- 互怼中带着喜欢
- 自嘲
- 金句适合引用

### 范例
```
【欢喜冤家】
A: "你是我见过最烦人的人。"
B: "谢谢，我也这么觉得你。"
（两人对视，观众知道有戏）

【尴尬场面】
A: "所以你喜欢的人是..."
B: "...不重要。反正不是你。"
（其实就是）

【闺蜜实话】
"你看他的眼神，跟看冰箱里的蛋糕一样。"
"你现在的状态，叫做嘴硬。"

【告白】
"我花了三十年才明白：讨厌一个人是很累的。所以我决定...喜欢你。"
"我不是个会说漂亮话的人。但我想跟你吃很多顿饭。"

【Grand Gesture】
"我追了三班地铁才赶上你。你能不能别走这么快？"
"我知道我搞砸了。但如果你愿意给我一次机会，我会用一辈子来证明我值得。"
```

## 结构设计

### Billy Mernit的7节拍
| 节拍 | 内容 | 比例 |
|------|------|------|
| 1. 设置 | 两人各自的世界 | 5-10% |
| 2. Meet-Cute | 可爱/有趣的相遇 | 10-15% |
| 3. 转折点1 | 开始发展关系 | 25% |
| 4. 热恋/上升 | 甜蜜时光/障碍出现 | 25-50% |
| 5. 转折点2 | 大危机/分手 | 50-60% |
| 6. 分离 | 黑暗时刻/各自反思 | 60-75% |
| 7. 重聚 | Grand Gesture/HE | 75-100% |

### Meet-Cute设计
- 相撞（经典）
- 误会对方是别人
- 被迫同处一室
- 网上认识线下见面
- 假扮情侣

### 障碍类型
- 阶级/身份差异
- 前任/情敌
- 事业vs爱情
- 家庭反对
- 地理距离
- 个人创伤

## 喜剧技巧

### 笑料来源
| 类型 | 例子 |
|------|------|
| 误会 | 说的A以为是B |
| 尴尬 | 被发现偷看/说错话 |
| 反差 | 霸道总裁怕狗 |
| 失败 | 精心计划全毁 |
| 配角 | 闺蜜的神助攻/猪队友 |

### 喜剧节奏
- 铺垫 → 铺垫 → 抖包袱
- Rule of 3：两次正常，第三次反转

---

## 视觉设计

### 场景
- 咖啡店（相遇/约会）
- 婚礼（重要场合）
- 机场/车站（分别/追逐）
- 屋顶/天台（浪漫时刻）
- 雨中（和解/告白）

### 色调
- 温暖、明亮
- 柔光
- 浪漫色彩：粉、暖黄、天蓝
