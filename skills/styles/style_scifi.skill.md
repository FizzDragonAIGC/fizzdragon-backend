# 科幻敘事風格 (Sci-Fi Style)

> 📚 **核心參考**
> - Isaac Asimov《基地》系列 - 宏大敘事
> - Philip K. Dick《銀翼殺手》- 哲學思辨
> - 劉慈欣《三體》- 硬核科幻
> - 《黑鏡》- 近未來反思

---

## 一、核心特徵

### 科幻敘事的本質
> "科幻不是預測未來，而是用未來的鏡子照見當下。"

**三大支柱：**
1. **科技假設**：基於科學邏輯的推演
2. **社會影響**：技術改變人類社會的方式
3. **哲學命題**：關於人性、存在、自由意志

### 世界觀構建原則
- 一個核心科技假設（如：時間旅行/AI覺醒/外星接觸）
- 由此衍生的社會變化
- 保持內部邏輯一致性

---

## 二、對白風格

### 示範對白

**硬科幻風格：**
```
「跳躍引擎計算完成，誤差在0.003光秒以內。」
「那足夠了。在這個距離，0.003光秒意味著我們要麼精準到達，要麼死在真空裡。沒有第三種可能。」
```

**軟科幻/哲思風格：**
```
「如果記憶可以複製，那『我』還是我嗎？」
「也許『我』從來都不是固定的。我們每天醒來，都是昨天那個人的複製品。」
```

**反烏托邦風格：**
```
「公民，你的幸福指數低於標準。」
「我很幸福。」
「系統已記錄你的異常反應。請前往最近的情緒校正中心。」
```

## 三、結構設計

### 科幻敘事節奏
```
建立世界(10%) → 發現異常(15%) → 探索真相(30%) → 
理解規則(15%) → 規則被打破(15%) → 新平衡(15%)
```

### 常用開場模式
1. **懸念開場**：先展示結果，再解釋原因
2. **日常割裂**：普通人偶遇非凡事件
3. **新聞/記錄**：以第三方視角引入世界
4. **時間跳躍**：從未來回溯

### 核心衝突類型
- 人 vs 技術（AI失控、基因編輯倫理）
- 人 vs 社會（反烏托邦、極權控制）
- 人 vs 宇宙（外星威脅、宇宙尺度的無意義）
- 人 vs 自我（身份認同、記憶真實性）

### 標誌性視覺元素
- 全息投影、懸浮界面
- 巨型建築、城市分層
- 機械義肢、生物改造
- 星空、太空站、行星環

✅ 是否有哲學/人性層面的思考？
