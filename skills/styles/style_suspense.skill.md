# 悬疑推理风格 (Suspense/Mystery Style)

> 伏笔密集、层层反转、公平推理、真相震撼

---

## 核心特征

### 推理规则
- **公平原则**：所有线索必须在揭示真相前展示给观众
- **红鲱鱼**：误导性线索，让观众猜错
- **不可能犯罪**：看似无法完成的谋杀
- **最终反转**：真相必须出人意料又合情合理

### 悬疑类型
| 类型 | 特点 | 代表作 |
|------|------|--------|
| 本格推理 | 公平解谜、逻辑严密 | 《名侦探柯南》|
| 社会派 | 关注动机、社会批判 | 《白夜行》|
| 心理悬疑 | 角色内心、不可靠叙述者 | 《消失的爱人》|
| 法庭剧 | 庭审对决、证据交锋 | 《十二怒汉》|

### 角色模板
| 角色 | 功能 |
|------|------|
| 侦探 | 揭示真相，观众代入点 |
| 凶手 | 隐藏在众人中，有动机 |
| 嫌疑人A | 有动机但无机会 |
| 嫌疑人B | 有机会但无动机 |
| 受害者 | 死前行为藏有线索 |
| 助手 | 提问让侦探解释，帮观众理解 |

## 参考方法论

### 希区柯克悬念理论
```
悬念 ≠ 惊吓

❌ 两人聊天 → 突然爆炸 = 惊吓（5秒）
✅ 观众知道桌下有炸弹 → 两人聊天 = 悬念（5分钟）
```

### 伏笔设计
- **埋伏笔**：自然融入，不引起注意
- **假伏笔**：误导观众往错误方向推理
- **收伏笔**：真相揭示时回顾所有伏笔

### 三层结构
```
表层故事：观众看到的事件
↓
隐藏线索：散布在细节中
↓
真相：最终揭示，颠覆认知
```

## 台词风格

### 特点
- 对话中藏线索
- 每句话可能有双重含义
- 审讯/质问场景紧张
- 揭示真相时的高光台词

### 范例
```
【藏线索的对话】
"那天晚上我一直在家..."
（其实说的是另一天）

【误导性台词】
"我和他从小一起长大，怎么可能杀他？"
（正因为如此才有杀人动机）

【侦探推理】
"等等...你刚才说'被害者的右手'，但我们从未提到是右手..."

【真相揭示】
"凶手...就在这个房间里！"
"从一开始，这就是一场精心设计的骗局。"
"那个人早在三年前就已经死了。"

【凶手独白】
"没有人会相信的...因为真相太残忍了。"
```

## 节奏设计

### 推理剧结构
| 阶段 | 比例 | 内容 |
|------|------|------|
| 发现 | 10% | 发现尸体/事件 |
| 调查 | 40% | 收集线索、访问证人 |
| 混乱 | 20% | 多个嫌疑人、互相矛盾 |
| 顿悟 | 10% | 关键线索触发 |
| 揭示 | 20% | 真相大白、伏笔回收 |

### 悬念节奏
```
谜题1 → 解开 → 更大谜题 → 假答案 → 真相反转
```

### 信息控制
- 给观众的信息 < 侦探知道的 = 惊喜
- 给观众的信息 > 侦探知道的 = 悬念
- 给观众的信息 = 侦探知道的 = 公平推理

## 视觉设计

### 场景氛围
- 雨夜、雾气、昏暗
- 密闭空间（庄园、岛屿、列车）
- 老宅、图书馆、档案室

### 镜头语言
- 特写关键证物
- 意味深长的眼神
- 回忆闪回的视觉风格统一
- 真相揭示时的蒙太奇

## 写作检查清单

✅ 所有线索在揭示前都展示了吗？（公平原则）
✅ 有足够的红鲱鱼吗？
✅ 真相出人意料又合情合理吗？
✅ 凶手的动机足够有说服力吗？
✅ 伏笔都回收了吗？
✅ 侦探的推理观众能跟上吗？
✅ 最终反转让人想重看吗？
