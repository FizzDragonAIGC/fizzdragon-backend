# 热血少年风格 (Shounen Style)

> 友情努力胜利、升级打怪、宿敌羁绊、燃点爆发

---

## 核心特征

### 三大核心
- **友情**：同伴羁绊是力量源泉
- **努力**：通过训练/修炼变强
- **胜利**：永不放弃最终获胜

### 角色模板
| 角色 | 设定 | 功能 |
|------|------|------|
| 主角 | 热血笨蛋/有隐藏天赋/家族血统 | 观众代入点 |
| 宿敌 | 天才/目标相同/亦敌亦友 | 推动主角成长 |
| 导师 | 传授绝招/有神秘过去 | 智慧传承 |
| 队友A | 智囊型/冷静分析 | 战术支持 |
| 队友B | 搞笑担当/忠诚 | 调节气氛 |
| 队友C | 女性角色/治愈系 | 情感线 |
| 最终BOSS | 绝对强大/有悲惨过去 | 终极挑战 |

### 升级系统
- 明确的力量等级（战斗力/境界/段位）
- 每次突破有仪式感
- 新招式有专属名字
- 极限时刻觉醒隐藏力量

---

## 参考方法论

### 少年JUMP公式
```
初战失败 → 特训修炼 → 再战获胜 → 更强敌人出现
    ↑__________________________|
```

### 战斗设计
1. **开局**：双方试探
2. **中盘**：各展绝招
3. **逆境**：主角陷入危机
4. **觉醒**：闪回/羁绊触发
5. **逆转**：新招式/极限爆发
6. **收尾**：敌人倒下/认可主角

### 燃点设计
- 喊出招式名
- 同伴的声援
- 回忆闪回（为了谁而战）
- 伤痕累累依然站起
- 最后一击的仪式感

---

## 台词风格

### 特点
- 热血宣言
- 招式名中二
- 对手互相认可
- 誓言/羁绊台词

### 范例
```
【战斗宣言】
"我要成为火影！"
"这里就是我的战场！"
"为了同伴，我绝不会输！"

【招式喊名】
"螺旋丸！"
"天马流星拳！"
"卍解！"

【羁绊台词】
"因为...我们是伙伴啊！"
"你的意志，我来继承！"
"只要你还在，我就不会倒下！"

【认可对手】
"你很强...但我更强！"
"下次再战，我会更强！"
"果然...你是我认可的对手！"

【极限觉醒】
"还没有结束...我还能站起来！"
"这份力量...是大家给我的！"
"超越极限，这就是我的忍道！"
```

## 节奏设计

### 篇章结构
| 篇章 | 功能 | 时长 |
|------|------|------|
| 序章 | 世界观/主角出场 | 3-5集 |
| 初期篇 | 组队/基础战斗 | 10-20集 |
| 修炼篇 | 特训/学新招 | 5-10集 |
| 大赛篇 | 比赛/排名 | 20-30集 |
| 危机篇 | 大反派登场 | 30-50集 |
| 最终篇 | 终极对决 | 10-20集 |

### 单集结构
```
前情回顾 (1分钟)
↓
战斗继续/新情节 (18分钟)
↓
高潮时刻/悬念 (3分钟)
↓
下集预告 (30秒)
```

### 燃点节奏
```
铺垫 → 铺垫 → 铺垫 → 燃！→ 缓 → 铺垫 → 铺垫 → 更燃！
```

## 写作检查清单

✅ 主角有明确的梦想/目标吗？
✅ 有升级打怪的成长线吗？
✅ 羁绊/友情线足够感人吗？
✅ 战斗有招式名和仪式感吗？
✅ 燃点时刻让人想站起来吗？
✅ 宿敌足够有魅力吗？
✅ 有"为了同伴"的时刻吗？
