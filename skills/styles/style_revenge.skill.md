# 复仇爽剧风格 (Revenge/Power Fantasy Style)

> 打脸爽感、步步为营、大仇得报、智商在线

---

## 核心特征

### 爽点设计
- **打脸**：让欺负主角的人遭报应
- **逆袭**：从底层一路上升
- **智斗**：用智慧碾压对手
- **装逼**：低调装高调打
- **认清现实**：让反派知道自己多蠢

### 角色模板
| 角色 | 设定 | 功能 |
|------|------|------|
| 主角 | 曾经受害/有隐藏实力/极度隐忍 | 复仇执行者 |
| 主要反派 | 曾经加害者/现在得意洋洋 | 最终打脸对象 |
| 帮凶1 | 反派的狗腿/落井下石 | 首先被收拾 |
| 帮凶2 | 墙头草/见风使舵 | 最后求饶 |
| 新盟友 | 被主角魅力吸引 | 助力复仇 |
| 旧友 | 当年没帮主角/愧疚 | 现在想帮忙 |

### 复仇阶段
```
受难 → 蛰伏 → 布局 → 收网 → 清算
```

---

## 参考方法论

### 《基督山伯爵》结构
1. **蒙冤**：无辜受害，失去一切
2. **觉醒**：在绝境中获得力量/财富/知识
3. **归来**：以新身份回归
4. **布局**：一个个接近仇人
5. **清算**：让仇人付出代价
6. **收场**：复仇完成后的空虚/救赎

### 《黑暗荣耀》技巧
- 不是简单杀死敌人，而是让他们体验同样的痛苦
- 用敌人自己的手段对付他们
- 观众知道计划但受害者不知道 = 悬念+爽感
- 留下道德灰度：复仇者也付出代价

## 台词风格

### 特点
- 金句要狠、要准
- 反派嚣张时更显蠢
- 主角台词冷而有力
- 打脸时刻要有仪式感

### 范例
```
【反派嚣张（铺垫打脸）】
"你以为你是谁？你什么都不是！"
"跪下来求我，也许我会考虑放过你。"
"你不过是个蝼蚁，我随时可以捏死你。"

【主角隐忍】
"是的，我什么都不是。" （内心OS：现在不是）
"我记住了。" （冷淡但观众知道有好戏看）

【反转揭示】
"你知道我是谁吗？"
"十年了，我等这一天等了十年。"
"你当年做的事，现在该还了。"

【打脸时刻】
"你以为我是蝼蚁？错了。蝼蚁是你。"
"跪下。" （简短有力）
"现在，你知道什么叫绝望了吗？"

【最终清算】
"我不会杀你。我要让你活着，看着你失去一切。"
"这才是你应得的结局。"
```

## 节奏设计

### 情绪曲线
```
压抑 → 小爽 → 压抑 → 中爽 → 更大压抑 → 大爽 → 终极压抑 → 终极爽
```

### 爽点密度
| 阶段 | 内容 | 爽感 |
|------|------|------|
| 前期 | 被欺负，观众积累愤怒 | 压抑 |
| 中前 | 收拾小喽啰 | ⭐⭐ |
| 中期 | 打脸中级boss | ⭐⭐⭐ |
| 中后 | 反派反扑，危机 | 压抑 |
| 高潮 | 终极打脸 | ⭐⭐⭐⭐⭐ |

### 单集结构
```
开场小爽（回顾上集打脸）
↓
新威胁/压抑
↓
主角布局
↓
执行计划
↓
爽点高潮
↓
新悬念钩子
```

## 关键技巧

### 铺垫打脸
- 反派越嚣张，打脸越爽
- 给足反派"作死"的时间
- 让观众恨透了再打脸

### 信息差
- 观众知道主角在布局
- 反派浑然不知
- 这种信息差创造悬念和期待

### 升级对手
- 小喽啰 → 中层 → boss → 幕后黑手
- 每个对手的"作恶程度"递增
- 打脸的爽感也递增

## 写作检查清单

✅ 主角受的委屈够惨吗？（观众愤怒值）
✅ 反派够可恨吗？
✅ 打脸的铺垫够足吗？
✅ 爽点密度够高吗？（每集至少一个小爽）
✅ 主角智商在线吗？（不是靠运气）
✅ 最终打脸够解气吗？
✅ 有没有留道德灰度？（不变成纯爽文）
