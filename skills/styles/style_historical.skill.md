# 歷史劇敘事風格 (Historical Drama)

> 📚 **核心參考**
> - 《大明王朝1566》- 權謀深度
> - 《羅馬》HBO - 史詩格局
> - 《唐頓莊園》- 時代細節
> - 二月河帝王系列 - 歷史質感

---

## 一、核心特徵

### 歷史劇的魅力
> "歷史是任人打扮的小姑娘，但好的歷史劇讓觀眾相信這是唯一的真相。"

**三大支柱：**
1. **時代質感**：服化道、語言、禮儀的還原
2. **人物命運**：在歷史洪流中的個人選擇
3. **權力博弈**：政治、經濟、軍事的交織

---

## 二、對白風格

### 示範對白

**朝堂論政：**
```
「陛下，此乃動搖國本之策！」
「國本？朕問你，是祖宗的規矩重要，還是百姓的命重要？」
「臣...臣不敢。」
「不敢說，還是不敢想？」
```

**民間煙火：**
```
「掌櫃的，今年的茶葉怎麼貴了三成？」
「客官有所不知，北邊打仗，運茶的商隊折了一半。這價錢，已是我賠本賣了。」
```

**戰場廝殺：**
```
「將軍，弟兄們三日沒吃飯了。」
「告訴他們，過了這座山，就是糧倉。過不去，骨頭就埋在這。」
```

## 三、時代語言規範

### 稱謂對照
| 現代 | 古代（官方） | 古代（民間） |
|------|-------------|-------------|
| 我 | 臣/微臣/本官 | 俺/小的/奴家 |
| 你 | 陛下/大人/將軍 | 客官/相公/娘子 |
| 父親 | 父皇/家父 | 爹/阿爺 |
| 好的 | 遵旨/謹遵 | 是/得令 |

### 語言禁忌
❌ 現代詞彙（OK、搞定、壓力大）
❌ 白話語氣詞（嗯啊、呃）
❌ 太過文言（之乎者也過多）
✅ 半文半白，朗朗上口

---

## 四、結構設計

### 歷史劇節奏
```
時代背景(5%) → 主角出場(10%) → 捲入風波(15%) → 
步步為營(30%) → 危機爆發(20%) → 塵埃落定(20%)
```

### 常用敘事手法
1. **多線並進**：朝廷、民間、戰場交織
2. **以小見大**：從小人物看大時代
3. **史實+虛構**：真實事件+合理想象
4. **今古對照**：暗示現代問題
