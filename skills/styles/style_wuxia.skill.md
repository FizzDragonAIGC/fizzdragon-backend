# 武侠江湖风格 (Wuxia Style)

> 侠义精神、江湖恩怨、绝世武功、快意恩仇

---

## 核心特征

### 精神内核
- **侠**：侠之大者，为国为民
- **义**：重义轻利，言出必行
- **情**：儿女情长，英雄气短
- **仇**：恩怨分明，有仇必报

### 世界观要素
| 要素 | 说明 |
|------|------|
| 门派 | 少林、武当、峨眉、丐帮... |
| 武功 | 内功、外功、轻功、暗器 |
| 江湖规矩 | 辈分、恩怨、比武、盟约 |
| 宝物 | 武林秘籍、神兵利器 |
| 身份 | 大侠、魔头、隐士、官府 |

### 角色原型
| 角色 | 设定 |
|------|------|
| 大侠 | 武功高强、品德高尚、为民请命 |
| 少侠 | 成长中的主角、有奇遇 |
| 魔头 | 野心勃勃、武功邪门 |
| 师父 | 传授武功、提供智慧 |
| 红颜 | 江湖儿女、亦柔亦刚 |
| 老前辈 | 隐世高人、关键时刻出手 |

## 台词风格

### 特点
- 文白夹杂，古意盎然
- 金句传世
- 武功招式有名
- 对话见性格

### 范例
```
【侠义】
"侠之大者，为国为民。"
"我辈练功习武，所为何事？"
"今日你我剑下见真章！"

【江湖规矩】
"出来混，迟早要还。"
"武林中人，向来一言既出，驷马难追。"
"前辈请了！晚辈有礼！"

【恩怨】
"此仇不报，枉为人子！"
"你我之间，已无话可说。"
"十年前的账，今日来清算。"

【情义】
"红颜知己，人生难求。"
"此生只负天下，不负卿。"
"江湖路远，望君珍重。"

【武功】
"接我一招'亢龙有悔'！"
"想不到阁下竟会'独孤九剑'..."
"此等内力，已臻化境！"
```

## 结构设计

### 金庸式长篇结构
```
卷一：出身（主角身世之谜）
卷二：奇遇（获得秘籍/拜师）
卷三：初露锋芒（小试牛刀）
卷四：江湖纷争（卷入恩怨）
卷五：危机（被陷害/重伤）
卷六：突破（武功大成）
卷七：真相（身世揭晓）
卷八：决战（最终对决）
卷九：归隐/传承
```

### 古龙式结构
- 悬疑开场（先有谜题）
- 浪子行走江湖
- 一个个谜团解开
- 最终真凶揭晓
- 诗意收尾

## 武打设计

### 招式命名
- 动物：龙爪、虎扑、蛇形、鹤翔
- 自然：风卷、雨打、雷霆、电闪
- 哲学：太极、两仪、四象、八卦
- 情感：断肠、销魂、夺命、伤心

### 打斗节奏
```
对峙（气势） → 试探（轻点） → 交锋（激烈） → 
绝招（高潮） → 胜负（结果） → 余波（后果）
```
