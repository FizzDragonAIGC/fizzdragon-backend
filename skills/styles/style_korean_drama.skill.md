# 韩剧风格 (Korean Drama Style)

> 情感细腻、财阀世界、命运纠葛、虐心但治愈

---

## 核心特征

### 情感设计
- **慢热铺垫**：前4集建立人设和关系
- **推拉节奏**：靠近-误会-分开-再靠近
- **虐心时刻**：让观众心碎但不放弃希望
- **治愈结局**：BE慎用，HE为主

### 世界观
- 财阀家族 vs 普通人家
- 命运般的相遇（小时候见过/前世因缘）
- 家族秘密/出身之谜
- 职场/校园作为舞台

### 角色设定
| 角色 | 设定模板 |
|------|----------|
| 男主 | 外冷内热/霸道总裁/温柔守护 |
| 女主 | 坚强乐观/有隐藏伤痛/善良正义 |
| 男二 | 暖男/比男主先出现/注定错过 |
| 女二 | 财阀女/男主前女友/亦正亦邪 |
| 恶婆婆 | 反对这段感情的长辈 |

## 参考方法论

### 韩剧16集经典结构
| 集数 | 功能 | 内容 |
|------|------|------|
| 1-2 | 相遇 | 命运般的第一次见面，产生误会 |
| 3-4 | 纠缠 | 被迫相处，从讨厌到好奇 |
| 5-6 | 心动 | 开始在意对方，内心挣扎 |
| 7-8 | 确认 | 告白/接受/第一次危机 |
| 9-10 | 甜蜜 | 短暂幸福时光 |
| 11-12 | 危机 | 身世秘密/误会/第三者 |
| 13-14 | 分离 | 被迫分开/虐心 |
| 15-16 | 重逢 | 解开误会/大结局 |

### 经典套路
- 👔 男主湿身/生病女主照顾
- 🌧️ 雨中告白/分手
- 🏥 意外住院探望
- 🚗 车祸/失忆（慎用）
- 👩‍👧 父母反对增加障碍
- 💍 假订婚/契约关系变真心

## 台词风格

### 特点
- 情感直接但不肉麻
- 金句多，适合截图传播
- 独白揭示内心
- 眼神交流替代语言

### 范例
```
经典台词模板：

【心动确认】
"为什么...我的心会这样？"

【虐心告白】
"我不是不喜欢你，是太喜欢你了，所以才要离开。"

【守护宣言】
"从现在起，我会站在你身边，不让任何人伤害你。"

【误会解开】
"原来那时候...你也是这样想的吗？"

【大结局】
"能遇见你，是我这辈子最幸运的事。"
```

### 对白节奏
- 重要对话要有"呼吸感"
- 沉默和眼神比说话更重要
- 回忆闪回配合当下对话

## 视觉/情绪节奏

### 场景设计
- 咖啡店/书店（相遇）
- 公司/学校（日常）
- 海边/屋顶（告白/重要对话）
- 机场/车站（分离/重逢）
- 家族宅邸（冲突）

### BGM配合
- 心动时刻：钢琴+弦乐
- 虐心时刻：慢版主题曲
- 甜蜜时刻：轻快吉他
- 紧张时刻：电子节拍

### 慢镜头时刻
- 第一次对视
- 意识到喜欢的瞬间
- 分别场景
- 最终拥抱

## 写作检查清单

✅ 男女主有命运般的连接点吗？

✅ 有足够的推拉和误会吗？
✅ 虐心程度观众能接受吗？
✅ 配角有自己的故事线吗？
