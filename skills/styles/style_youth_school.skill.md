# 青春校園敘事風格 (Youth/School Style)

> 📚 **核心參考**
> - 《那些年》- 純愛回憶
> - 《花樣年華》- 情緒氛圍
> - 《垫底辣妹》- 成長熱血
> - 《我的少女時代》- 懷舊青春

---

## 一、核心特徵

### 青春敘事的本質
> "青春故事的核心不是年齡，而是那種相信一切皆有可能的心態。"

**三大支柱：**
1. **成長蛻變**：從迷茫到找到方向
2. **情感萌動**：友情、愛情、師生情
3. **夢想追逐**：考試、比賽、社團

---

## 二、對白風格

### 示範對白

**暗戀獨白：**
```
「每次下課鈴響，我都會不自覺地看向窗外的籃球場。
不是喜歡籃球。
是他在那裡。」
```

**友情衝突：**
```
「你變了。」
「我沒變，是你太執著於我們十五歲的樣子。」
「那時候的我們，不是更好嗎？」
「那時候的我們，什麼都不懂。」
```

**勵志對話：**
```
「老師，我可能考不上了。」
「考不考得上是結果，但努不努力是選擇。你現在能選的，只有後者。」
```

---

## 三、結構設計

### 青春敘事節奏
```
日常建立(15%) → 羈絆形成(20%) → 
目標出現(15%) → 挫折低谷(20%) → 
成長蛻變(20%) → 告別/新開始(10%)
```

### 常用情節元素
- 🎵 學園祭/畢業典禮
- 🏃 社團活動/比賽
- 📝 考試/升學壓力
- 💕 告白/暗戀/誤會
- 👥 轉學生/新學期
- 🌸 櫻花/夕陽/天台

---

## 四、情緒設計

### 青春特有情緒
| 情緒 | 表現方式 |
|------|----------|
| 心動 | 心跳加速、臉紅、說不出話 |
| 嫉妒 | 偷看、酸言酸語、故意疏遠 |
| 迷茫 | 走神、發呆看天空、耳機音樂 |
| 熱血 | 吶喊、奔跑、揮灑汗水 |
| 離別 | 欲言又止、背對流淚、揮手 |

### 氛圍營造
- **春天**：新的開始、相遇
- **夏天**：熱血、汗水、社團
- **秋天**：文化祭、成長、變化
- **冬天**：考試、告白、離別

---

## 五、視覺設計

### 色調建議
| 場景 | 主色調 | 氛圍 |
|------|--------|------|
| 教室 | 陽光暖黃 | 青春溫暖 |
| 天台 | 天空藍 | 自由開闘 |
| 夕陽 | 橙紅 | 懷舊感動 |
| 夜晚 | 路燈暖黃 | 曖昧浪漫 |
| 雨天 | 灰藍 | 憂傷思念 |

### 標誌性畫面
- 陽光穿過窗戶的教室
- 跑道上奔跑的身影
- 天台上並肩坐著
- 騎自行車的放學路
- 夏日祭的煙火

## 七、寫作清單

✅ 角色是否有成長弧線？
✅ 情感是否含蓄不過度？
✅ 是否有懷舊/時代感元素？
✅ 對白是否符合學生口吻？
✅ 是否有標誌性青春場景？
✅ 結局是否餘韻悠長？
✅ 是否避免了太過成人的內容？
