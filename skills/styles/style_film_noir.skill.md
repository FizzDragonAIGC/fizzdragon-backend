# 黑色电影风格 (Film Noir Style)

> 低光照明、蛇蝎美人、宿命论、道德灰色地带

---

## 核心特征

### 视觉标志
- **低调打光**：强烈阴影、百叶窗光栅
- **倾斜构图**：不安、失衡
- **雨夜街道**：反射、孤独
- **烟雾缭绕**：神秘、危险

### 叙事特点
- 倒叙/闪回开场（主角已经完蛋）
- 第一人称旁白（cynical、疲惫）
- 命运无法逃脱
- 没有真正的好人

### 角色原型
| 角色 | 设定 |
|------|------|
| 侦探/主角 | 愤世嫉俗、有原则但被生活打败 |
| 蛇蝎美人 (Femme Fatale) | 致命诱惑、利用男人达到目的 |
| 雇主/委托人 | 隐藏真实动机 |
| 硬汉 | 打手、暴力执行者 |
| 腐败警察 | 与黑暗势力勾结 |

## 台词风格

### 特点
- 简洁、尖锐、带刺
- 讽刺、厌世
- 隐喻多、不直说
- 旁白揭示内心

### 范例
```
【旁白开场】
"她走进我的办公室的那一刻，我就知道她是麻烦。
但麻烦从来不会提前告诉你它值多少钱。"

【硬汉对白】
"你不喜欢我的方式？太糟糕了。这是唯一的方式。"
"我不是个好人，但我还没坏到那个地步。"
"每个人都有价码。区别在于货币。"

【蛇蝎美人】
"你会帮我的，对吗？"（眼泪欲滴）
"我只是一个被困住的女人..."（全是谎言）
"相信我。"（这三个字是最大的红旗）

【宿命感】
"我知道这会怎么结束。每个人都知道。但我们还是走了进去。"
"有些错误，你在犯的时候就知道会后悔。但你还是会犯。"
```

## 结构设计

### 经典结构
```
开场：主角旁白，已知结局（倒叙）
↓
回忆：事情是怎么开始的
↓
委托：神秘美女出现
↓
调查：每个线索都指向更深的黑暗
↓
背叛：信任的人是敌人
↓
收网：真相揭示
↓
结局：主角付出代价（死亡/孤独/道德沦丧）
```

### 节奏
- 开头慢、气氛重
- 中间逐渐紧张
- 暴力爆发突然、残酷
- 结尾悲凉

## 视觉设计

### 打光
| 技术 | 效果 |
|------|------|
| 低调打光 | 阴影占主导 |
| 硬光源 | 清晰阴影边缘 |
| 百叶窗光 | 囚笼隐喻 |
| 单光源 | 孤独感 |

### 场景
- 私家侦探办公室（破旧、烟雾）
- 夜间街道（雨、霓虹）
- 酒吧/夜总会
- 豪宅（隐藏罪恶）
- 廉价旅馆

### 色调
- 黑白或低饱和度
- 如有彩色：暗调、冷色
- 红色：唯一的暖色，代表危险/情欲

✅ 有宿命感的旁白吗？
✅ 主角是cynical的吗？
✅ 有蛇蝎美人或类似角色吗？
✅ 视觉上够黑暗吗？
✅ 结局是苦涩的吗？
✅ 没有真正的好人吗？
✅ 对白够尖锐吗？
