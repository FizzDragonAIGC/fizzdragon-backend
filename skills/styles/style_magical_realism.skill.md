# 魔幻现实主义风格 (Magical Realism Style)

> 魔幻与日常并存、不解释超自然、家族史诗、时间循环

---

## 核心特征

### 精神内核
- **魔幻即日常**：超自然事件发生，角色习以为常
- **不解释**：黄蝴蝶满天飞，没人问为什么
- **循环**：命运/名字/诅咒代代相传
- **孤独**：人与人之间无法真正沟通

### 叙事特点
| 特点 | 说明 |
|------|------|
| 全知视角 | 上帝俯瞰家族命运 |
| 时间跳跃 | 过去/现在/未来穿插 |
| 预言/诅咒 | 已知结局仍要走向它 |
| 重复 | 名字、命运、错误代代重复 |
| 隐喻 | 每个意象都有象征意义 |

### 魔幻元素设计
- 不是奇幻/fantasy（没有魔法体系）
- 魔幻发生在日常场景
- 角色不惊讶
- 读者也慢慢接受

## 台词风格

### 特点
- 诗意、意象丰富
- 预言式陈述
- 接受荒诞
- 时间感模糊

### 范例
```
【魔幻即日常】
"那天下午，奥雷里亚诺上校第一次见到了冰块。"
"黄蝴蝶总是跟着他，从出生到死亡。"
"自从丈夫死后，她就一直在织寿衣，织完就拆，拆完再织。"

【预言/诅咒】
"家族中第一个人被绑在树上，最后一个人被蚂蚁吃掉。"
"所有叫奥雷里亚诺的人都会孤独终老。"
"我们注定要重复父辈的错误。"

【时间感】
"许多年以后，面对行刑队，他会想起父亲带他去看冰块的那个遥远的下午。"
"她等了五十三年七个月零十一天。"

【孤独】
"他们各自用不同的方式孤独着。"
"世界太新，很多事物还没有名字。"
"我们都是被遗忘的人，在被遗忘的地方。"
```

## 结构设计

### 家族史诗结构
```
第一代：创建/诅咒起源
↓
第二代：扩张/第一次重复
↓
第三代：鼎盛/矛盾积累
↓
第四代：分裂/命运重演
↓
第五代：衰落/预言应验
↓
第六代（可选）：终结/循环打破或完成
```

### 时间叙事
- 可以从任何一代开始
- 过去/现在不断交织
- 用"许多年后"/"当年"跳跃
- 结局可能是开头

### 魔幻元素分布
- 不要集中在一处
- 自然地融入日常描写
- 前后呼应
- 有象征意义

## 意象设计

### 常见象征
| 意象 | 含义 |
|------|------|
| 黄蝴蝶 | 命运、死亡预兆 |
| 雨 | 悲伤、时间流逝 |
| 房子 | 家族、衰败 |
| 镜子 | 过去、记忆 |
| 植物生长 | 生命力、自然秩序 |
| 鬼魂 | 过去的重量 |

### 颜色
- 黄色：命运、死亡
- 红色：激情、暴力
- 绿色：生命、遗忘
- 白色：纯洁、逝去

✅ 魔幻元素自然融入了吗？

✅ 有家族/时间循环感吗？
✅ 有预言/诅咒吗？
✅ 意象有象征意义吗？
✅ 语言够诗意吗？
✅ 孤独感贯穿全文吗？
