# 好莱坞古典叙事风格 (Hollywood Classic Style)

> 三幕剧结构、因果逻辑清晰、个人英雄主义、大团圆结局

---

## 核心特征

### 结构规则
- **三幕比例**：25% - 50% - 25%
- **激励事件**：第一幕10-15%处发生
- **中点反转**：第二幕50%处必有大转折
- **最低点**：第二幕75%处主角跌入谷底
- **高潮**：第三幕85-95%最终对决

### 情节要素
- 主角有清晰的Want（外在目标）和Need（内在需求）
- 每场戏必须推动情节或揭示角色
- 冲突逐步升级，赌注越来越高
- 结局解决Want，实现Need

### 角色设计
- **主角**：有缺陷但可认同
- **导师**：提供智慧或工具
- **反派**：是主角的黑暗镜像
- **爱情线**：B故事承载主题

## 参考书籍精华

### Syd Field《Screenplay》
- 范式结构：第一幕建置→第二幕对抗→第三幕解决
- Plot Point 1 (第25-27页)：进入特殊世界
- Plot Point 2 (第85-90页)：最终决战准备

### Blake Snyder《Save the Cat》15节拍
| 节拍 | 位置 | 功能 |
|------|------|------|
| Opening Image | 1% | 主角原始状态 |
| Theme Stated | 5% | 主题暗示 |
| Set-up | 1-10% | 世界观建立 |
| Catalyst | 12% | 激励事件 |
| Debate | 12-25% | 主角犹豫 |
| Break into 2 | 25% | 进入新世界 |
| B Story | 30% | 爱情/友情线启动 |
| Fun & Games | 30-50% | 类型承诺兑现 |
| Midpoint | 50% | 假胜利或假失败 |
| Bad Guys Close In | 50-75% | 压力升级 |
| All Is Lost | 75% | 最低点 |
| Dark Night of Soul | 75-80% | 主角反思 |
| Break into 3 | 80% | 顿悟时刻 |
| Finale | 80-99% | 最终对决 |
| Final Image | 100% | 改变后的主角 |

### Robert McKee《Story》
- 价值转换：每场戏必须有正负极转换
- 鸿沟理论：期望与结果之间的差距创造戏剧性
- 激励事件：打破生活平衡的那一刻

### Christopher Vogler《The Writer's Journey》
- 12阶段英雄之旅（见narrative_complete.skill）
- 8种原型角色

## 台词风格

### 特点
- 对白服务于情节推进
- 潜台词比明文重要
- 每个角色有独特声音
- 金句设计在关键节点

### 范例
```
经典结构：
角色A：陈述问题
角色B：提出障碍  
角色A：揭示内心
角色B：给出选择
（沉默或动作）
角色A：做出决定
```

### 避免
- ❌ 角色直接说出主题
- ❌ 过度解释情节
- ❌ 所有角色说话方式相同
- ❌ 没有冲突的闲聊

---

## 节奏控制

### 场景长度
- **对话戏**：2-3页（2-3分钟）
- **动作戏**：1-2页
- **情感高潮**：可延长到5页

### 张弛有度
```
紧张 → 舒缓 → 更紧张 → 舒缓 → 最紧张 → 释放
```

## 写作检查清单

✅ 激励事件在前15%发生了吗？
✅ 主角有清晰的Want和Need吗？
✅ 中点有反转吗？
✅ All Is Lost够低吗？
✅ 高潮解决了核心冲突吗？
✅ 最终画面与开场画面呼应吗？
✅ 每场戏都有价值转换吗？
