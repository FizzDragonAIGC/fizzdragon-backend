# Core Methodology Skill (融合精华版)

## 🎯 敘事結構 (McKee + Blake Snyder)

### 三幕結構
- **起 (25%)**: 建立世界、角色、核心衝突
- **承轉 (50%)**: 遞進障礙、中點逆轉、最低谷
- **合 (25%)**: 高潮對決、主題揭示、新平衡

### 15節拍 (Save the Cat)
1. 開場畫面 → 2. 主題陳述 → 3. 鋪墊 → 4. 催化劑 → 5. 辯論
6. 進入第二幕 → 7. B故事 → 8. 遊戲時間 → 9. 中點 → 10. 壞人逼近
11. 一無所有 → 12. 靈魂黑夜 → 13. 進入第三幕 → 14. 結局 → 15. 終場畫面

### 每集鉤子設計
- **開頭鉤**: 30秒內抓住注意力
- **中點懸念**: 劇情反轉或新信息
- **結尾鉤**: 未解決的衝突，讓觀眾想看下集

## 👤 角色塑造 (千面英雄 + 角色弧線)

### 核心三層
| 層次 | 說明 | 例子 |
|------|------|------|
| WANT | 外在目標 | 找到寶藏 |
| NEED | 內在需求 | 學會信任 |
| FLAW | 致命缺陷 | 過度自負 |

### 角色弧線類型
- **正向弧**: 缺陷→認知→改變→成長
- **負向弧**: 拒絕改變→墮落
- **平弧**: 堅持信念改變世界

### 英雄旅程12階段
平凡世界 → 歷險召喚 → 拒絕召喚 → 遇見導師 → 跨越門檻 → 
考驗盟友敵人 → 進入洞穴 → 嚴峻考驗 → 獲得獎賞 → 
返回之路 → 復活 → 帶著萬靈丹歸來

## 📝 劇本格式

### 場景標題
```
【場景】描述 - 時間
```

### 對話格式
```
角色名
（動作/情緒）台詞內容
```

### 動作描寫
- 用現在式
- 具體可視化
- 每段≤4行
