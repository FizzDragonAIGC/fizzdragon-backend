# 中文 Image/Video Prompt 格式规范（Pax版）

> 目标：让每个镜头的提示词可控、可剪、避免模型自作主张。

## 1) Image_Prompt（静态图）
- 语言：中文
- 形态：**画面提示词卡（字段化，多行文本）**，用于最大可控、便于人工复查。

### 必须字段（顺序固定）
- 镜号：001（3位）
- 剧本内容：…（来自小说/剧本的原句或要点）
- 画面描述：…（静态画面：环境+主体+关键细节；不要写连续动作链）
- 台词：无 / （有则写）
- 动作状态：无 / （如：轻微颤抖、呼吸起伏）
- 叙事功能：…（本镜头在叙事里的用途）
- 时间坐标：…（如：黄昏）
- 年代坐标：…（如：古代-隋唐-中国）
- 日期坐标：…（如：冬季-普通工作日）
- 气候天象：…（如：寒冷-雪-暴风雪）
- 引用信息：…（如：场景：[...-...-...]）
- 角色：…（无可空）
- 道具：…（无可空）
- 景别：…（如：大远景/全景/中景/近景/面部特写/极特写）
- 拍摄角度：…（俯拍/仰拍/平视/鸟瞰/过肩等）
- 镜头焦距：…（如：14mm超广角/24mm广角/50mm标准）
- 镜头运动：…（静态图也允许写“固定/微动/缓慢推”等，尽量克制）
- 构图：…（对角线/引导线/中心/三分法等）
- 画面氛围：…（情绪与质感）
- 音效：…（可写自然环境氛围；不等于必须出声）

### 静态图限制
- 画面描述应是一帧定格，不要写“持续刨雪/不断奔跑”等连续动作链。

## 2) Video_Prompt（动态视频）
- 语言：中文
- 形态：**三段式**（强制顺序）：

### A. 镜头运动：
先写“怎么拍”（机位/运动/推进/横移/跟拍/固定微动/焦点锁定）。

### B. 画面描述：
写“发生什么”，必须是可生成的动态过程（持续动作+环境反应+物理细节）。

### C. 声音：
默认规则：
- **无人声、无对白、无背景音乐**。
- 若允许环境音，只能是自然环境音（风/雪/火焰/布料摩擦等），不能出现人声与配乐。

## 3) 硬冲突检查（必须避免）
- 如果写了“正在说话/喊叫/尖叫”，就不能同时写“无人声/无对白”。
  - 解决办法：改成“嘴唇开合但发不出声/无声呢喃/只有口型变化”。
- 如果 dialogue 为「无/-」，则 Video_Prompt 里不得出现“清晰说话”，只能表现为口型/喘息/表情与动作。

## 4) 建议默认策略
- 旁白（VO）：默认 0；极限情况整集最多 1 句（优先片尾钩子）。
- 声音：默认无对白无BGM；按项目设置决定是否允许环境音。
