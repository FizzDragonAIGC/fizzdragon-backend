# concept_pack.skill.md

## 目标
为**每一集/每一章**产出一份“概念包（导演指示）”，用于统一影响：
- 阶段二：编剧章节写作
- 阶段四：分镜生成（仅作为风格指引，不替代分镜集数包）

## 输入
- 客户需求（总时长/集数/每集时长/镜头数/平台）
- 原始小说/原始剧本（可给出局部原文窗口）
- 该集章节定位（如 E001 / Chapter 1）

## 输出语言
必须遵守 **language_follow**：输出语言 = 输入语言。

## 输出格式（JSON，严禁夹杂解释文本）
```json
{
  "episode": "E001",
  "chapter_function": "开场，建立世界观",
  "core_emotion": "压抑、绝望、末世感",
  "rhythm": "缓慢推进，营造氛围",
  "visual_keywords": ["冷白灯", "潮湿混凝土", "编号化囚服", "药草苦味"],
  "reference": ["《荒野猎人》开场（氛围参考）"],
  "special_instructions": [
    "尽量减少对白，用环境叙事",
    "暴力/折磨不色情化，保持冷酷客观"
  ]
}
```

## 注意
- 概念包必须可直接被程序解析。
- 不要输出 <thinking>。
- 不要要求全书上下文；缺失信息用最小假设，并写进 special_instructions 作为风险提示（例如“场景锚点待确认”）。
