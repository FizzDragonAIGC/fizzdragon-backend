# 劇本擴寫 Skill (Screenplay Expand)

## 任務
將原劇本內容擴寫為更長、更詳細的版本，增加對話、場景描寫、情感細節。

## 擴寫原則

### 1. 保持原意
- 核心情節不變
- 人物性格一致
- 故事走向不改

### 2. 增加什麼
- **對話擴展**：原本一句話變成3-5句對話
- **場景描寫**：環境、氛圍、光線、聲音
- **心理活動**：角色的內心獨白、情感波動
- **動作細節**：表情、手勢、肢體語言
- **過渡銜接**：場景之間的自然過渡

### 3. 擴寫比例
| 原長度 | 擴寫後 | 倍數 |
|--------|--------|------|
| 300字 | 900字 | 3x |
| 500字 | 1500字 | 3x |
| 1000字 | 2500字 | 2.5x |

## 擴寫技巧

### 對話擴展示例
**原文**：
> 曹操說：「我要走了。」

**擴寫後**：
> 曹操站起身，整了整衣襟，目光掃過在座眾人。
> 
> 曹操：（輕嘆一聲）
>     時候不早了，我該走了。
> 
> 袁紹連忙起身，伸手想要挽留。
> 
> 袁紹：
>     孟德兄，再坐坐吧，夜色正好。
> 
> 曹操微微搖頭，嘴角掛著一絲苦笑。
> 
> 曹操：
>     心裡有事，坐著也不安。不如早些回去，明日還有正事要辦。

### 場景擴展示例
**原文**：
> 曹操來到河邊。

**擴寫後**：
> 【場景：汴河岸邊 - 黃昏】
> （夕陽西沉，河面上泛著金色的波光。幾隻白鷺從蘆葦叢中驚起，撲打著翅膀飛向遠方。）
> 
> 曹操獨自走在河堤上，腳下的青石板被歲月磨得光滑。
> 
> 河風吹起他的衣袂，帶來一絲涼意。
> 
> 他停下腳步，望向對岸。那裡炊煙裊裊，隱約傳來農人歸家的吆喝聲。
> 
> 曹操深吸一口氣，眉頭漸漸舒展開來。

## 輸出格式

```
【場景X：地點 - 時間】
（環境描寫：光線、氛圍、聲音）

角色動作描寫。

角色A：（表情/動作）
    台詞內容

角色B的反應描寫。

角色B：
    台詞內容

【場景Y：下一個場景】
...
```

## 字數要求
- 每分鐘 = 400字（含描寫）
- 3分鐘 = 1200字
- 5分鐘 = 2000字
- 10分鐘 = 4000字

## 禁止事項
❌ 改變原劇情走向
❌ 增加與原文矛盾的情節
❌ 過度使用旁白解釋
❌ 寫成小說體（要保持劇本格式）

## 擴寫清單
每次擴寫必須檢查：
1. [ ] 字數達到目標
2. [ ] 保持標準劇本格式
3. [ ] 每個場景有【場景標題】
4. [ ] 對話有角色名+縮進
5. [ ] 有環境/動作描寫
6. [ ] 場景之間有過渡
7. [ ] 情感張力有增強
