# 敘事結構完整技能 (Narrative Complete)

> 📚 **融合6大敘事理論**
> - Joseph Campbell《千面英雄》- 英雄之旅
> - Vladimir Propp《故事形態學》- 31功能

> - Pamela Douglas《電視劇寫作》- 劇集結構

## 一、三幕式核心結構

```
第一幕 (25%) - 建置
├── 普通世界 → 激勵事件 → 拒絕召喚 → 遇見導師
└── 【關鍵】跨越第一道門檻

第二幕 (50%) - 對抗  
├── 考驗/盟友/敵人 → 深入洞穴 → 核心磨難
├── 【中點】獲得獎賞/假勝利
└── 【關鍵】最黑暗時刻

第三幕 (25%) - 解決
├── 回程 → 復活 → 攜帶靈丹歸來
└── 【關鍵】高潮對決 + 新平衡
```

## 四、故事格子診斷 (Coyne)

### 五大組件 (每場景必備)
```
1. 激勵事件 → 打破平衡
2. 複雜化 → 壓力升級
3. 危機 → 二選一困境
4. 高潮 → 不可逆選擇
5. 解決 → 新平衡
```

| 類型 | 正極 | 負極 | 核心問題 |

| 愛情 | 愛 | 恨 | 能在一起嗎？|
| 成長 | 成熟 | 幼稚 | 能覺醒嗎？|
| 犯罪 | 正義 | 不義 | 真相是什麼？|
| 恐怖 | 生存 | 詛咒 | 能逃脫嗎？|
| 世界觀 | 信仰 | 幻滅 | 意義何在？|

## 五、改編技巧 (Seger)

### 三大策略
| 策略 | 技巧 | 範例 |
|------|------|------|
| **濃縮** | 刪除副線 | 《魔戒》刪Tom Bombadil |
| | 合併角色 | 三朋友→一閨蜜 |
| | 壓縮時間 | 數月→數天 |
| **擴展** | 增加副線 | 配角獨立故事 |
| | 深化背景 | 擴展世界觀 |
| | 增加衝突 | 添加障礙 |
| **翻譯** | 內心→行動 | 獨白→表情/動作 |
| | 描述→視覺 | 文字→攝影/美術 |

### 字數→集數公式
```
小說字數 / 15,000 = 基礎集數 (每集10-15分鐘)
調整: 對話密集+15%, 動作+20%, 內心描寫-10%
```

## 六、劇集結構 (TV Series)

### 類型對比
| 類型 | 定義 | 範例 |
|------|------|------|
| 單元劇 | 每集獨立 | CSI |
| 連續劇 | 跨集連貫 | 權力遊戲 |
| 混合型 | 單集+長線 | X檔案 |

### 單集結構 (10-15分鐘)
```
開場鉤子    → 30秒-1分鐘
情境建立    → 2-3分鐘
複雜化      → 4-5分鐘
危機決策    → 2-3分鐘
高潮+鉤子   → 2-3分鐘
```

### 多線敘事
| 線 | 權重 | 功能 |
|----|------|------|
| A線 | 55% | 主要故事 |
| B線 | 30% | 角色發展 |
| C線 | 15% | 伏筆/調節 |

### 10集季度弧線
```
第1-2集: 引擎啟動 (激勵事件)
第3-4集: 複雜化 (提高賭注)
第5集: 中點反轉 (大揭示)
第6-8集: 危機 (最低點)
第9-10集: 高潮決戰
```

### 章末鉤子類型
| 類型 | 說明 | 範例 |
|------|------|------|
| 懸崖 | 生死中斷 | 槍響畫面黑 |
| 揭露 | 驚人真相 | 「我是你父親」|
| 決定 | 重大抉擇前 | 要不要開槍 |
| 反轉 | 顛覆認知 | 死人活了 |
| 情感 | 情緒高點 | 分手/告白 |

## 八、鉤子設計 (Cliffhangers)

| 類型 | 說明 | 範例 |
|------|------|------|
| **懸念鉤** | 未解之謎 | 「他到底是誰？」|
| **危機鉤** | 危險即將發生 | 定時炸彈倒計時 |
| **揭示鉤** | 驚人真相 | 「他是你父親」|
| **情感鉤** | 關係轉折 | 分手/告白 |
| **死亡鉤** | 角色命運未定 | 槍響，畫面變黑 |

**每集結尾必須有鉤子！** 讓觀眾想看下一集。

## 九、起承轉合 (Phase) 分配

| Phase | 集數範圍 | 功能 | 情緒曲線 |
|-------|----------|------|----------|
| **起** | 前25%集數 | 世界觀建立、角色登場、日常展示 | 平穩→好奇 |
| **承** | 25%-50%集數 | 矛盾發展、關係深化、第一次危機 | 緊張上升 |
| **轉** | 50%-75%集數 | 衝突升級、危機爆發、關鍵抉擇 | 高度緊張 |
| **合** | 後25%集數 | 高潮對決、真相揭曉、大結局 | 爆發→釋放 |
