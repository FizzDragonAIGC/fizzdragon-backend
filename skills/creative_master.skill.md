# 創作大師技能 (Creative Master)

> 融合：narrative + character + dialogue + music + voiceover
> 精簡重複，保留創作核心

---

## 一、敘事結構 (Narrative)

### 三幕結構
| 幕 | 比例 | 功能 | 節點 |
|----|------|------|------|
| 第一幕 | 25% | 建置 | 激勵事件(12%) |
| 第二幕 | 50% | 對抗 | 中點(50%)、低谷(75%) |
| 第三幕 | 25% | 解決 | 高潮(90%)、結局 |

### 情節節拍 (Blake Snyder)
1. 開場(1%) - 展示世界
2. 主題(5%) - 暗示主題
3. 鋪墊(1-10%) - 日常生活
4. 催化劑(12%) - 打破平衡
5. 辯論(12-25%) - 猶豫掙扎
6. 第二幕開始(25%) - 踏入新世界
7. B故事(30%) - 副線/愛情
8. 遊樂場(30-50%) - 新世界探索
9. 中點(50%) - 假勝利/假失敗
10. 壞人逼近(50-75%) - 壓力升級
11. 失去一切(75%) - 最低谷
12. 靈魂黑夜(75-80%) - 絕望反思
13. 第三幕(80%) - 找到答案
14. 結局(80-99%) - 最終對決
15. 終幕(99-100%) - 新平衡

### 章節節奏
```
起 (25%): 建立+激勵，每集1個鉤子
承 (25%): 升級+探索，每集1個反轉
轉 (25%): 對抗+危機，每集1個衝突
合 (25%): 高潮+解決，每集1個答案
```

## 二、角色設計 (Character)

### 角色維度
| 維度 | 說明 | 範例 |
|------|------|------|
| WANT | 外在目標 | 找到兇手 |
| NEED | 內在需求 | 學會信任 |
| LIE | 錯誤信念 | 只能靠自己 |
| GHOST | 背景創傷 | 被朋友背叛 |
| FLAW | 性格缺陷 | 過度控制 |

### 角色弧線
```
開始: LIE支配行為
中點: LIE受到挑戰
低谷: LIE崩塌
高潮: 接受NEED，改變
結局: 新的平衡
```

### 角色類型
| 類型 | 功能 | 範例 |
|------|------|------|
| 主角 | 推動故事 | 視角人物 |
| 對手 | 製造障礙 | 價值觀對立 |
| 導師 | 提供智慧 | 引導成長 |
| 盟友 | 支持主角 | 情感支撐 |
| 變形者 | 忠誠不定 | 製造懸念 |
| 門檻守衛 | 考驗主角 | 階段Boss |

## 三、台詞設計 (Dialogue)

### 台詞功能
| 功能 | 範例 |
|------|------|
| 推進劇情 | 「他們今晚動手。」 |
| 揭示角色 | 「我從不求人。」 |
| 製造衝突 | 「你根本不懂！」 |
| 建立關係 | 「你何時學會說謊？」 |
| 埋設伏筆 | 「這鑰匙，收好。」 |

### 潛台詞技巧
```
表層: 說的話
潛層: 真正意思

範例:
說：「天氣不錯。」
意：「我不想聊這個。」
```

### 角色語言指紋
| 特徵 | 說明 |
|------|------|
| 口頭禪 | 專屬用語 |
| 句式 | 長句/短句/碎片 |
| 詞彙 | 文雅/粗俗/專業 |
| 節奏 | 快/慢/停頓 |
| 禁忌 | 絕不說的話 |

## 四、音樂設計 (Music)

### 情緒映射
| 情緒 | 調性 | 節奏 | 樂器 |
|------|------|------|------|
| 緊張懸疑 | 小調 | 不規則 | 弦樂顫音 |
| 史詩壯闘 | 大調 | 進行曲 | 銅管合唱 |
| 浪漫溫情 | 大調 | 舒緩 | 鋼琴弦樂 |
| 悲傷哀愁 | 小調 | 緩慢 | 大提琴 |
| 恐怖驚悚 | 不協和 | 突變 | 尖銳音效 |
| 動作追逐 | 小調 | 快速 | 電子打擊 |

### 音樂節拍
```
動作場景: 音樂 = 剪輯節奏
對話場景: 音樂退後
情感高潮: 音樂主導
```

## 五、配音設計 (Voiceover)

### 情緒參數
| 情緒 | 語速 | 音調 | 音量 | 特點 |
|------|------|------|------|------|
| 平靜 | 正常 | 中 | 中 | 平穩 |
| 憤怒 | 快 | 高 | 大 | 咬字重 |
| 悲傷 | 慢 | 低 | 小 | 顫抖停頓 |
| 恐懼 | 快碎 | 高 | 小→大 | 氣息不穩 |
| 興奮 | 快 | 高 | 大 | 起伏大 |

### 配音類型
| 類型 | 用途 |
|------|------|
| V.O. 畫外音 | 旁白敘述 |
| Inner Voice | 內心獨白 |
| ADR | 後期配音 |

## 六、場景設計

### 場景功能
| 類型 | 功能 | 密度 |
|------|------|------|
| 開場 | 建立世界 | 低 |
| 日常 | 展示角色 | 低 |
| 對話 | 推進關係 | 中 |
| 轉折 | 改變方向 | 高 |
| 高潮 | 情感爆發 | 最高 |
| 收尾 | 解決餘韻 | 中低 |

### 場景結構
```
每場景 = 目標 + 障礙 + 結果
結果必須改變局勢(+/-)
```

## 七、衝突設計

### 衝突層次
| 層次 | 說明 | 範例 |
|------|------|------|
| 內心 | 自我掙扎 | 理智vs慾望 |
| 人際 | 角色對立 | 主角vs對手 |
| 社會 | 制度壓迫 | 個人vs體制 |
| 自然 | 環境考驗 | 生存危機 |
| 命運 | 不可抗力 | 疾病/死亡 |

### 衝突升級
```
第一幕: 小衝突，試探
第二幕前: 中衝突，升級
中點: 假解決
第二幕後: 大衝突，危機
第三幕: 終極衝突，決戰
```

## 八、情感設計

### 情感弧線
```
開始: 情感A (如：孤獨)
中點: 情感轉變 (如：連結)
低谷: 情感B (如：失去)
高潮: 情感爆發 (如：犧牲)
結局: 新情感 (如：釋懷)
```

### 觀眾情緒節奏
```
張力曲線:
起: ▂▃ (建立期待)
承: ▄▅▆ (逐漸升溫)
轉: ▇█▇ (高潮爆發)
合: ▅▃▂ (餘韻回落)
```
