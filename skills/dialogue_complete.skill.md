## 📊 專業研究與數據支撐

### 學術研究來源
| 研究/作者 | 年份 | 核心發現 |
|----------|------|---------|
| **John F. Burrows《Computation into Criticism》** | 1987 | 風格計量學奠基之作，證明角色可通過功能詞頻率數學區分 |
| **Labov 社會語言學研究** | 1966-2010 | 語言變化從中下階層發起並擴散，方言特徵可量化追蹤 |
| **Writers Guild of America 調查** | 2024 | 50+編劇訪談：2010年代後對語言真實性要求顯著提高 |
| **Journal of Screenwriting 風格分析** | 2019 | 對《大吉嶺限號快車》對白的計量分析，證明Wes Anderson角色聲音可量化區分 |

### 劇本對白數據標準
| 指標 | 數值 | 來源 |
|------|------|------|
| 平均劇本總詞數 | **~9,000 詞** | 好萊塢劇本統計 |
| 每分鐘對白詞數 | **125 詞**（範圍50-200） | Tameri Guide |
| 每頁平均詞數 | **165-180 詞** = 緊湊寫作 | r/Screenwriting |
| 1頁 ≈ 1分鐘銀幕時間 | 行業標準 | 所有編劇手冊 |

### 對白真實性研究 (The Conversation, 2025)
> **調查發現**：50+位美國編劇工會成員表示，2010年代以來對語言代表性的敏感度顯著提升。
> 
> **關鍵趨勢**：
> - 更重視聘請能代表角色聲音和背景的編劇
> - 「真實性」(Authenticity) 被視為對白設計的首要考量
> - 避免使用語音拼寫(phonetic spelling)來標記低階層角色

### 風格計量學 (Stylometry) 應用
基於 **Burrows' Delta 方法**：
- 通過分析功能詞（如「的」「了」「嗎」）頻率區分角色
- 角色語言習慣可量化為「語言指紋」
- 用於驗證角色聲音是否足夠獨特

## 一、台詞功能分類

| 功能 | 說明 | 範例 |
|------|------|------|
| 推進劇情 | 傳遞關鍵信息 | 「他們今晚就會動手。」 |
| 揭示角色 | 展現性格/背景 | 「我這輩子從沒求過人。」 |
| 製造衝突 | 對立觀點碰撞 | 「你根本不懂！」 |
| 建立關係 | 角色互動張力 | 「你什麼時候學會說謊的？」 |
| 埋設伏筆 | 看似無心的暗示 | 「這把鑰匙，你收好。」 |
| 製造笑料 | 喜劇效果 | 機智反諷、誤會對話 |
| 抒發情感 | 情緒宣洩 | 「我只是...太累了。」 |

## 二、台詞層次設計

### 表層 vs 潛台詞
```
表層（說的話）: 「今天天氣不錯。」
潛台詞（真實意圖）: 「我不想談這個話題。」
```

### 潛台詞技巧
| 技巧 | 說明 | 範例 |
|------|------|------|
| 迴避 | 不正面回答 | 「你餓不餓？我去做飯。」 |
| 轉移 | 話題轉向 | 「說起這個，你媽最近怎麼樣？」 |
| 沉默 | 不說比說更多 | （長久的沉默） |
| 反話 | 說相反的意思 | 「隨便你，我無所謂。」 |
| 雙關 | 一語雙關 | 「這盤棋，你想怎麼走？」 |

## 三、對白節奏

### 節奏類型
| 類型 | 特點 | 適用場景 |
|------|------|---------|
| 快速交鋒 | 短句、快接 | 爭吵、對峙 |
| 緩慢沉重 | 長停頓、慢語速 | 悲傷、告白 |
| 自然流暢 | 日常語感 | 生活場景 |
| 詩意獨白 | 文學性強 | 內心戲、旁白 |

### Sorkin式對白（快速機智）
```
A: 你知道這意味著什麼嗎？
B: 我知道。
A: 你不知道。
B: 我知道我不知道，但我知道你會告訴我。
A: 這意味著我們完了。
B: ...好吧，那我確實不知道。
```

### Tarantino式對白（閒聊中見張力）
```
A: 你知道法國麥當勞的四分之一磅漢堡叫什麼嗎？
B: 不知道。
A: 他們叫它皇家芝士堡。
B: 為什麼？
A: 因為他們用公制。
```

## 👤 根據人物小傳設計台詞

> **📊 研究支撐**
> - **Paddy Chayefsky** (3座奧斯卡原創劇本獎)：角色心理深度決定對白張力
> - **Billy Wilder** (3座奧斯卡)：Sharp dialogue源於角色核心矛盾
> - **Francis Ford Coppola**《教父》研究：複雜角色需要多層次潛台詞

### 人物小傳關鍵要素 → 台詞風格

| 小傳要素 | 影響台詞的方面 | 範例 |
|----------|---------------|------|
| **教育程度** | 詞彙量、句式複雜度 | 博士vs小學學歷 |
| **職業背景** | 專業術語、思維方式 | 醫生、律師、農民 |
| **性格特質** | 說話直接度、情緒表達 | 內向vs外向 |
| **原生家庭** | 禮儀習慣、情感表達 | 嚴厲vs溺愛 |
| **創傷經歷** | 迴避話題、防禦反應 | 戰爭、失親 |
| **年齡** | 流行語、價值觀 | Z世代vs50後 |
| **地域文化** | 方言、習慣用語 | 見上表 |
| **社會階層** | 說話姿態、用詞 | 貴族vs底層 |

### 人物小傳 → 台詞設計流程

```
1. 閱讀人物小傳
   ↓
2. 提取關鍵特徵：國籍、職業、性格、創傷、口頭禪
   ↓
3. 確定語言風格：正式/口語、長句/短句、直接/委婉
   ↓
4. 設計專屬口頭禪和說話習慣
   ↓
5. 根據場景情緒調整語氣
```

### 範例：同一場景，不同人物

**場景：被問到過去**

**富家小姐（養尊處優、迴避）**
> 「......那些事啊？都過去了嘛。來，嚐嚐這個，剛從巴黎帶回來的。」

**退伍軍人（創傷、壓抑）**
> （長久沉默）「......沒什麼好說的。」（轉身離開）

**香港滑板少女（叛逆、直接）**
> 「關你咩事？問那麼多做乜？」

**日本武士（克制、禮節）**
> 「失禮了。此事...恕在下不便言及。」

**美國牛仔（直率、豪爽）**
> "Look, I don't like talkin' about the past. What's done is done."

## 五、對白禁忌

### ❌ 避免
- **過度解釋**: 不要讓角色說出觀眾已經知道的
- **廣播對白**: 「作為你的哥哥，我必須告訴你...」
- **書面語**: 真實對話不會這麼完整通順
- **空洞感嘆**: 「天哪！」「不會吧！」

### ✅ 推薦
- **打斷**: 真實對話常被打斷
- **重疊**: 多人同時說話
- **口誤**: 說錯/改口
- **停頓**: ...（沉默的張力）

## ⚡ 快速檢查清單

✅ **每個角色是否有獨特語言指紋？**（遮住名字能認出誰說的）
✅ **潛台詞是否比表面意思更豐富？**
✅ **對白是否推進劇情或揭示角色？**
✅ **地域/職業/教育特徵是否一致？**
✅ **避免了廣播式解說？**（角色不會說「作為你哥哥...」）
✅ **節奏變化是否配合情緒？**
