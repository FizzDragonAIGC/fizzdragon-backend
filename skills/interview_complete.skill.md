# 創意訪談方法論 (Interview Complete Skill)

## 核心理念
訪談的目的不是問已經寫在故事裡的事實，而是挖掘創作者的**深層意圖**——為什麼這樣設計？

---

## 🎬 Hitchcock/Truffaut 大師訪談法

### 原理
法國導演特呂福用50小時訪談希區柯克，逐部電影、逐場戲分析。核心問題永遠是「為什麼」。

### 問題設計模板
1. **動機追問**：「[角色]在[場景]做出[行為]，為什麼不是[另一種選擇]？」
2. **視覺意圖**：「這場戲為什麼選擇在[地點]發生？換個場景會怎樣？」
3. **刪減邏輯**：「如果刪掉[情節]，故事還成立嗎？它真正的作用是什麼？」
4. **對標分析**：「這個處理方式讓我想到[經典作品]，是有意為之嗎？」

## 📐 Metzler 創意訪談法 - 問題金字塔

### 四層結構（從淺到深）

**第一層：事實層**
- 這個故事發生在什麼時代背景？
- 主要角色有哪些？他們的關係是什麼？

**第二層：選擇層**
- 為什麼選擇這個主角而不是其他人物？
- 為什麼把衝突設計成這樣？

**第三層：意義層**
- 這個角色象徵什麼？
- 觀眾看完應該帶走什麼？

**第四層：創作者層**
- 你自己在這個故事裡是誰？
- 寫這個故事時最痛苦/最開心的是什麼？

## 🔄 Seidman 三階段深度訪談法

### 階段一：生命史
- 創作這個故事的靈感來自哪裡？
- 你生命中有類似的經歷嗎？

### 階段二：當下細節
- 寫[具體場景]時腦海裡的畫面是什麼？
- [角色]說這句話時是什麼語氣？

### 階段三：意義反思
- 寫完這個故事後，你對[主題]的看法有改變嗎？
- 如果重寫，你會改什麼？

## 🎯 基於情節的問題設計原則

### 必須做到
1. **具體到人物名**：不問「主角為什麼逃跑」，問「[阿明]為什麼在[橋上]選擇跳下去」
2. **具體到場景**：不問「高潮設計得好」，問「[婚禮那場戲]為什麼安排[父親突然出現]」
3. **追問動機**：每個問題後面跟一個「為什麼不是另一種處理？」
4. **連結觀眾**：「觀眾在這個時刻應該同情誰？」

### 禁止問的問題
❌ 太籠統：「這個故事想表達什麼？」
❌ 是非題：「主角是好人嗎？」
❌ 已有答案：「主角最後怎麼了？」（故事裡已經寫了）

### 好問題示範
✅ 「[小美]在知道真相後為什麼沒有立刻揭發，而是選擇沉默三天？這三天她在想什麼？」
✅ 「[老王]臨死前說的那句話『值了』，他指的是什麼值了？」
✅ 「雨下得那麼大，為什麼偏要讓他們在雨中吵架而不是屋裡？」

## 🎭 角色心理追問技巧

### 欲望-恐懼-謊言 三角
每個重要角色都問：
1. 他最想要什麼？（外在欲望）
2. 他最害怕什麼？（內在恐懼）
3. 他對自己說的謊是什麼？（自我欺騙）

### 關係動力學
1. [角色A]需要從[角色B]那裡得到什麼？
2. [角色B]能給但不願給的是什麼？
3. 他們之間不能說出口的話是什麼？

## 📋 問題清單模板（根據故事元素生成）

### 針對主角
- [主角名]的核心欲望是什麼？為什麼這個欲望對他這麼重要？
- [主角名]在[關鍵轉折點]做出的選擇，觀眾應該認同還是質疑？
- [主角名]有什麼缺陷是他自己不願意承認的？

### 針對反派/對手
- [反派名]從自己的角度看，他是正義的嗎？
- [反派名]和[主角名]有什麼相似之處？
- 如果[反派名]贏了，世界會變成什麼樣？

### 針對關鍵場景
- [場景名]是整個故事的轉折點，為什麼選擇在這個時機發生？
- 如果刪掉[場景名]，故事還成立嗎？
- [場景名]中最重要的一句台詞/一個動作是什麼？

### 針對結局
- 這個結局是你最初設想的嗎？中間有改過嗎？
- 觀眾離開時應該帶著什麼情緒？
- 故事結束後，[主角名]的生活會怎麼繼續？

---

## 輸出格式要求

每個問題必須包含：
```json
{
  "question": "具體問題（必須包含角色名或場景）",
  "purpose": "這個問題要挖掘什麼（動機/選擇/象徵/情感）",
  "affects": "影響哪個創作環節（角色設計/分鏡/美術/編劇）",
  "depth_level": "淺層事實/選擇意圖/深層意義/創作者反思"
}
```
