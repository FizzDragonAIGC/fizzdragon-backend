# language_follow.skill.md

## 目标
在整个工作流中强制执行：**输出语言 = 输入语言**（不自动翻译、不混杂）。

## 适用范围
- 概念包（Concept Pack）
- 剧本包（Script Pack，含原文摘录 + 改写剧本）
- 分镜集数包（Storyboard Episode Pack，含该集剧本）
- 分镜逐镜头输出（含 Video_Prompt）

## 规则（硬约束）
1. **语言跟随**：你收到的主要输入内容是什么语言（中文/英文/其他），你就用同一种语言输出。
2. **禁止自动翻译**：除非用户明确要求“翻译/中英对照”。
3. **禁止中英混写**：输出不得中英夹杂（除非输入本身就是混合语言，或用户明确要求混合）。
4. **字段名不改变**：如果系统要求固定字段名/字段顺序（例如“镜号：/剧本内容：/画面描述：...”），字段名保持既定语言与形式；字段值语言跟随输入。

## 输出前自检
- [ ] 我是否在任何段落中无意插入了另一种语言？
- [ ] Video_Prompt 的“镜头运动/画面描述/声音”是否与输入语言一致？
- [ ] 是否出现多余的英文摄影术语（如输入为中文且未被用户要求）？
