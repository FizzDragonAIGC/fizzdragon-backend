## 一、Binge-Worthy 六大法則

### 1️⃣ 鉤子設計（HOOK）
每集結尾必須有強鉤子，讓觀眾無法停止觀看：

| 鉤子類型 | 說明 | 範例 |
|---------|------|------|
| 懸念型 | 關鍵問題未解答 | "他到底是不是兇手？" |
| 反轉型 | 顛覆認知 | 信任的人竟是叛徒 |
| 危機型 | 角色陷入絕境 | 被困、被追殺、時間耗盡 |
| 情感型 | 情感爆發點 | 分手、背叛、告白 |
| 揭示型 | 重大信息曝光 | 身世之謎、陰謀揭露 |

**前三集鉤子必須最強！** 這決定觀眾是否繼續。

### 2️⃣ 開場5分鐘定生死
Netflix數據：觀眾在**前5分鐘**決定是否繼續。

**開場必須包含：**
- 視覺衝擊（不是平淡日常）
- 核心衝突暗示
- 主角出場並展現性格
- 一個讓人好奇的問題

**❌ 錯誤開場**：主角起床、吃早餐、去上班...
**✅ 正確開場**：主角在做一件極端/有趣/危險的事

### 3️⃣ 不解決衝突（對比傳統單元劇）
傳統單元劇：每集有問題 → 解決 → 收尾
流媒體劇：問題累積 → 部分解決 → 新問題產生

**每集結構：**
- 回顧上集懸念（30秒內）
- 推進主線 + 引入新衝突
- 解決一個小問題，留下更大問題
- 結尾鉤子

### 4️⃣ 多線敘事交織
同時運行3-5條故事線：
- **A線**：主角主要目標（每集都推進）
- **B線**：主角情感線（每2-3集推進）
- **C線**：反派/對手線（神秘推進）
- **D線**：配角支線（調劑節奏）

技巧：在高潮前切換線路，製造懸念。

### 5️⃣ 角色弧線跨季設計
不要在一集內完成角色轉變！

**季度弧線設計：**
- 第1-3集：建立角色+缺陷
- 第4-6集：面對挑戰+掙扎
- 第7-9集：低谷+覺醒
- 第10集：蛻變但留遺憾（為下季鋪墊）

### 6️⃣ 節奏壓縮
流媒體比傳統TV更快：
- **場景更短**：平均2-3分鐘/場景
- **對白更精**：沒有廢話
- **畫面敘事**：能用畫面說的不用對白
- **每5分鐘一個小高潮**

## 二、Netflix劇本結構模板

### 45分鐘劇集結構
```
00:00-05:00 【冷開場/鉤子】
  - 視覺衝擊開場
  - 回顧上集懸念
  - 本集核心問題拋出

05:00-10:00 【回歸日常/鋪墊】
  - 角色狀態更新
  - 新線索/新角色引入
  - 暗示即將到來的麻煩

10:00-20:00 【衝突升級】
  - 主角嘗試解決問題
  - 遭遇阻礙
  - B線推進
  - 第一個小反轉

20:00-30:00 【中點轉折】
  - 情況急劇變化
  - 主角被迫改變策略
  - C線爆點
  - 情感衝突加深

30:00-40:00 【黑暗時刻/升溫】
  - 一切似乎要失敗
  - 內心掙扎
  - 多線交匯
  - 準備最終對決

40:00-45:00 【高潮+鉤子】
  - 本集目標達成/失敗
  - 但引出更大問題
  - 結尾鉤子（必須強！）
```

## 三、對白寫作法則

### ❌ 文藝腔（避免）
```
「我望著窗外的雨，思緒萬千。或許，這就是命運的安排吧。」
「人生如夢，夢如人生，誰又能說得清呢？」
```

### ✅ 流媒體風格
```
「說重點。」
「你知道這意味著什麼嗎？我們完了。」
「給我三分鐘。三分鐘後，要麼他死，要麼我們死。」
```

### 對白黃金法則：
1. **潛台詞 > 直白**：角色不會直接說出感受
2. **衝突 > 和諧**：每段對話都有張力
3. **行動 > 廢話**：說完就要做
4. **個性 > 通用**：每個角色說話方式不同
5. **短句 > 長句**：現代人注意力短

### 對白長度控制
- 普通對話：1-2句/次
- 情感爆發：3-4句/次
- 獨白/演講：極少使用，每集最多1次

## 四、場景節奏控制

### 場景長度建議
| 場景類型 | 建議長度 | 說明 |
|---------|---------|------|
| 動作場景 | 1-3分鐘 | 快切、緊張 |
| 對話場景 | 2-4分鐘 | 信息量大、有衝突 |
| 情感場景 | 3-5分鐘 | 可以稍慢，但要有層次 |
| 過渡場景 | 30秒-1分鐘 | 能省則省 |

### 場景切換技巧
- **硬切**：緊張感、時間緊迫
- **對比切**：從歡樂切到悲傷
- **聲音橋接**：下個場景的聲音提前進入
- **視覺匹配**：相似構圖連接不同場景

## 五、每集必備元素清單

✅ 開場鉤子（前5分鐘內）
✅ 至少1個反轉
✅ 主角面臨困境
✅ 情感時刻（觀眾產生共鳴）
✅ 推進至少2條故事線
✅ 結尾鉤子
✅ 下集預告暗示

## 六、成功案例分析

### 《魷魚遊戲》
- 開場鉤子：巴掌換錢的震撼場景
- 每集結尾：淘汰懸念、背叛揭露
- 節奏：遊戲段落快，人性段落慢
- 對白：簡短、直接、帶情緒

### 《乘風破浪的姐姐》式敘事
- 懸念：誰會被淘汰？
- 情感：選手故事引發共鳴
- 節奏：緊張+溫情交替

### 《紙房子》
- 開場：從結局開始（倒敘懸念）
- 多線：8個人物線同時推進
- 鉤子：每集都有計劃變動

## 七、輸出要求

寫劇本時必須：
1. **開場5分鐘**必須有視覺衝擊+懸念
2. **對白精煉**：刪掉所有廢話
3. **每5分鐘**安排一個小高潮
4. **結尾必須**有強鉤子
5. **避免文藝腔**：用行動和對話推進
