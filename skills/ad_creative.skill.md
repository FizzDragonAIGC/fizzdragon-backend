## 🎯 核心理念

### 廣告分鏡三要素
1. **策略先行**：賣點、受眾、情緒定位
2. **視覺敘事**：每個鏡頭都是一個故事
3. **文案點睛**：簡潔有力，行動召喚

## 📊 鏡頭拆解規則

### 15秒廣告 (6鏡頭)
| 鏡號 | 功能 | 平均時長 |
|------|------|---------|
| SC-01 | 引入/吸引注意 | 2秒 |
| SC-02 | 動作/場景 | 2.5秒 |
| SC-03 | 產品賣點 | 2.5秒 |
| SC-04 | 用戶反應 | 2.5秒 |
| SC-05 | 品牌展示 | 2.5秒 |
| SC-06 | 結尾召喚 | 3秒 |

### 30秒廣告 (8-10鏡頭)
- 更豐富的場景鋪陳
- 可加入次要角色
- 情緒起伏更完整
- 平均3-3.5秒/鏡

### 60秒廣告 (12-18鏡頭)
- 完整故事線
- 多場景切換
- 情感鋪墊更充分
- 平均3-5秒/鏡

## 💡 策略分析框架

### 賣點提煉法
1. **功能賣點**：產品做什麼
2. **情感賣點**：產品讓人感覺如何
3. **差異賣點**：與競品的區別
4. **場景賣點**：最佳使用場景

### 受眾畫像構建
- 年齡範圍
- 性別傾向
- 生活方式
- 痛點需求
- 消費能力
- 媒體偏好

### 情緒關鍵詞庫
| 類別 | 關鍵詞 |
|------|--------|
| 溫暖系 | 治愈、溫馨、安心、舒適 |
| 活力系 | 活力、自信、青春、動感 |
| 高級系 | 優雅、奢華、品味、格調 |
| 潮流系 | 酷炫、個性、前衛、創新 |

## ✍️ 文案創作原則

### Slogan設計
- 不超過10個字
- 朗朗上口
- 有記憶點
- 體現品牌調性

### 鏡頭文案
- 每句不超過15字
- 與畫面節奏匹配
- 情感層層遞進
- 最後句引導行動

### 文案類型
1. **旁白**：畫外音敘述
2. **對話**：角色台詞
3. **字幕**：屏幕文字
4. **Slogan**：品牌口號

## 🎨 視覺風格參考

### 溫暖治愈風
- 暖色調（橙、黃、棕）
- 自然光/逆光
- 慢節奏剪輯
- 柔和轉場

### 都市時尚風
- 冷暖對比
- 霓虹燈光
- 快節奏剪輯
- 動感轉場

### 自然清新風
- 綠色調/藍色調
- 日光/晨光
- 舒緩節奏
- 淡入淡出

### 高端奢華風
- 黑金/深色調
- 聚光燈/輪廓光
- 優雅節奏
- 精緻轉場

## 📋 廣告類型模板

### 產品功能廣告
```
痛點場景 → 產品出現 → 功能演示 → 效果展示 → 品牌收尾
```

### 情感故事廣告
```
角色介紹 → 情感鋪墊 → 產品融入 → 情感高潮 → 品牌升華
```

### 場景應用廣告
```
場景建立 → 需求產生 → 產品使用 → 場景改善 → 品牌關聯
```

### 對比廣告
```
痛點對比 → 舊方案問題 → 新產品優勢 → 效果對比 → 品牌承諾
```

## 🔄 迭代修改流程

### 修改定位
- 精確到具體鏡頭
- 明確修改維度（畫面/文案/節奏）
- 保持整體一致性

### 修改範圍
1. 單鏡頭調整：僅改指定鏡頭
2. 段落調整：連續2-3個鏡頭
3. 整體調整：風格/節奏重構
